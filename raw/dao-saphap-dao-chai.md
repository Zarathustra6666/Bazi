# ดาวสภาพ (伤官) และดาวใช้ (食神) ในปาจื้อ

## ความหมายพื้นฐาน

สองดาวนี้อยู่ในกลุ่ม "Output Stars" หรือดาวที่แสดงออก — เป็นพลังงานที่เจ้าชะตาส่งออกไปสู่โลก

## ดาวสภาพ (伤官 Shang Guan — Hurting Officer)

- **ชื่อ**: ดาวสภาพ / ดาวบาดเจ็บนาย
- **ธาตุ**: ธาตุเดียวกับเจ้าชะตา ขั้วตรงข้าม (Same element, opposite polarity as DM) แต่ควบคุม 正官
- **ลักษณะ**: ความคิดสร้างสรรค์สุดขีด ชอบฝืนกฎ ปฏิเสธอำนาจ กล้าแหวกแนว
- **ข้อควรระวัง**: ถ้าเด่นเกินไปโดยไม่มีตัวควบคุม อาจก่อปัญหากับผู้บังคับบัญชา
- **งานที่เหมาะ**: ศิลปะ นักเขียน นักดนตรี นักประดิษฐ์ สตาร์ทอัพ

## ดาวใช้ (食神 Shi Shen — Eating God / เทพโภคา)

- **ชื่อ**: ดาวใช้ / เทพโภคา / Food God
- **ธาตุ**: ธาตุที่เจ้าชะตาสร้าง (Same polarity as DM)
- **ลักษณะ**: ความสุข ความเพลิดเพลิน ความคิดสร้างสรรค์ที่มีความสุขใจ ชอบอาหาร ศิลปะ งานไอเดีย
- **จุดเด่น**: "มีศีลธรรมและความใจดี" นำมาซึ่งเครือข่ายและโอกาสรายได้
- **งานที่เหมาะ**: ธุรกิจอาหาร กราฟิก คอนเทนต์ การเขียน ยูทูบ ศิลปะการแสดง

## ความแตกต่างระหว่างสองดาว

| | ดาวสภาพ (伤官) | ดาวใช้ (食神) |
|---|---|---|
| ทิศทาง | ก้าวร้าว ฝืนกฎ | สงบ เพลิดเพลิน |
| ความคิดสร้างสรรค์ | ปฏิวัติ แหวกแนว | ศิลปะ มีความสุข |
| ความสัมพันธ์กับอำนาจ | ต่อต้าน (ทำร้าย 正官) | เป็นกลาง |
| โชคทรัพย์ | ผ่านดาวเงิน (偏财) | ผ่านดาวเงิน (偏财) |

## หลักการอ่านใน Chart

- ถ้า ดาวสภาพ/ดาวใช้ เด่นในดวง → บุคลิกสร้างสรรค์ชัดเจน
- ถ้ามีทั้งสองดาว → มีทั้งความเป็นศิลปินและความสนุกสนาน
- ต้องดูร่วมกับดาวเฉียบ (正官) ว่าถูกทำร้ายหรือไม่
