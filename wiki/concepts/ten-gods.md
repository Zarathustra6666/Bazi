# สิบเทพ (十神)

> สิบความสัมพันธ์เชิงพฤติกรรมที่อธิบายว่าแต่ละ element ในดวงชะตาสัมพันธ์กับ [[ธาตุเจ้าชะตา (Day Master)]] อย่างไร

## Overview
สิบเทพ (十神 Shí Shén, Ten Gods หรือ Ten Deities) คือระบบจำแนก 10 ความสัมพันธ์ระหว่าง element ใดๆ ในดวงชะตากับ [[ธาตุเจ้าชะตา (Day Master)]] ไม่ใช่เทพเจ้าตามตัวอักษร แต่เป็น **Behavioral Archetypes** — บุคลิกภาพ ความสัมพันธ์ และด้านชีวิตที่ element แต่ละตัวแทนในดวงของบุคคลนั้น Ten Gods เป็นภาษากลางของการตีความ [[Bazi (八字)]] และเป็นหัวใจของ Ziping Method (子平法)

## Origin & History (ที่มา)
ระบบ Ten Gods ปรากฏอย่างเป็นระบบตั้งแต่คัมภีร์ [[渊海子平 Yuan Hai Zi Ping]] สมัยราชวงศ์ซ่ง และได้รับการจัดระบบอย่างสมบูรณ์ใน [[子平真诠 Zi Ping Zhen Quan]] โดย Shen Xiaozhan แห่งราชวงศ์ชิง ซึ่งนำ Ten Gods มาผูกกับระบบ [[โครงสร้างดวงชะตา (格局)]] อย่างแน่นแฟ้น Xu Ziping (สมัยซ่ง) เป็นผู้กำหนดให้ Day Master เป็นจุดศูนย์กลาง ซึ่งทำให้ Ten Gods มีความหมายในบริบทของแต่ละบุคคล

## Core Definition (นิยามหลัก)

### หลักการกำหนด Ten Gods
Ten Gods ถูกกำหนดจาก 2 ปัจจัย:
1. **ความสัมพันธ์ 5 ธาตุ** ระหว่าง element นั้นกับ [[ธาตุเจ้าชะตา (Day Master)]]
2. **ความตรงหรือต่างขั้ว Yin-Yang** ระหว่างทั้งคู่

### ตารางสิบเทพครบ

| # | ชื่อจีน | ชื่ออังกฤษ | ความหมายหลัก | ความสัมพันธ์กับ Day Master |
|---|---------|-----------|-------------|--------------------------|
| 1 | 比肩 Bǐ Jiān | Companion / Friend | ตัวตน, ความมั่นใจ, ความเท่าเทียม | Same element, **same polarity** |
| 2 | 劫财 Jié Cái | Rob Wealth / Rival | การแข่งขัน, การสูญเสีย | Same element, **opposite polarity** |
| 3 | 食神 Shí Shén | Eating God / Talent | ความคิดสร้างสรรค์, ความสุข | Element DM produces, **same polarity** |
| 4 | 伤官 Shāng Guān | Hurting Officer / Maverick | ความฉลาดเฉียบแหลม, กบฏ, ศิลปะ | Element DM produces, **opposite polarity** |
| 5 | 偏财 Piān Cái | Indirect Wealth / Venturer | ทรัพย์สินผจญภัย, บิดา (ดวงหญิง) | Element DM controls, **opposite polarity** |
| 6 | 正财 Zhèng Cái | Direct Wealth / Earner | ทรัพย์สินมั่นคง, ภรรยา (ดวงชาย) | Element DM controls, **same polarity** |
| 7 | 七杀 Qī Shā | Seven Killings / Warlord | อำนาจ, ความกดดัน, ความเสี่ยง | Element controls DM, **opposite polarity** |
| 8 | 正官 Zhèng Guān | Direct Officer / Executive | สถานะ, กฎ, ชื่อเสียง | Element controls DM, **same polarity** |
| 9 | 偏印 Piān Yìn | Indirect Resource / Mystic | ความรู้นอกกระแส, ลัทธิ | Element generates DM, **opposite polarity** |
| 10 | 正印 Zhèng Yìn | Direct Resource / Mentor | การสนับสนุน, การศึกษา, มารดา | Element generates DM, **same polarity** |

## Five Elements Relationship (ความสัมพันธ์ 五行)

5 กลุ่มตาม Element Relationship + Yin-Yang:

| กลุ่ม | Ten Gods | ความสัมพันธ์กับ DM | บทบาทในชีวิต |
|-------|----------|-----------------|-------------|
| Companion Stars | 比肩, 劫财 | Same element as DM | พี่น้อง เพื่อน คู่แข่ง |
| Output Stars | 食神, 伤官 | DM produces | ลูก (ดวงหญิง), การแสดงออก, ทักษะ |
| Wealth Stars | 偏财, 正财 | DM controls | ทรัพย์สิน, พ่อ, ภรรยา (ดวงชาย) |
| Authority Stars | 七杀, 正官 | Controls DM | สามี (ดวงหญิง), เจ้านาย, กฎหมาย |
| Resource Stars | 偏印, 正印 | Generates DM | แม่, ที่ปรึกษา, การศึกษา |

## Interaction Rules (กฎปฏิสัมพันธ์)

**กฎ Polarity:**
- ตัวเลข**คี่** (1,3,5,7,9) = same polarity → ชื่อขึ้นต้น "正" (Direct/Zheng)
- ตัวเลข**คู่** (2,4,6,8,10) = opposite polarity → ชื่อขึ้นต้น "偏" (Indirect/Pian)

**Month Branch Ten God = 格局 (Chart Pattern):**
Ten God ที่ปรากฏใน Month Branch คือ pattern หลักของดวงชะตา — ดู [[โครงสร้างดวงชะตา (格局)]]

**Quality vs Quantity:**
- ไม่ใช่แค่นับว่ามีกี่ตัว แต่ดูว่าแต่ละตัว "ทำงาน" ได้ดีแค่ไหน
- Ten God ที่แข็งแกร่งและไม่ถูกรบกวน = ทรัพยากรที่ใช้งานได้จริง
- Ten God ที่ถูก clash/combine = ทำงานได้ไม่เต็มที่

**ดูตาม Pillar:**
| เสา | ด้านชีวิต |
|-----|----------|
| เสาปี | สังคม / บรรพบุรุษ |
| เสาเดือน | อาชีพ / เจ้านาย |
| เสาวัน | ตัวตน / คู่ครอง |
| เสาชั่วโมง | ลูก / อนาคต |

## Interpretation Methods (วิธีตีความ)

**ขั้นตอนการอ่าน Ten Gods:**
1. ระบุ [[ธาตุเจ้าชะตา (Day Master)]] จากกิ่งฟ้าเสาวัน
2. คำนวณ Ten God ของทุก element ในดวง (กิ่งฟ้าและ Hidden Stems)
3. ดู Ten God ใน Month Branch → นี่คือ[[โครงสร้างดวงชะตา (格局)|Pattern]]หลัก
4. ประเมินว่า Ten God แต่ละตัว "ทำงาน" ได้จริงหรือถูกรบกวน
5. ระบุ [[เทพใช้งาน (用神)]] ที่ "fulfill" Pattern

## Schools of Thought (สำนักต่างๆ)

**Classical Zi Ping:**
- Ten Gods วิเคราะห์ผ่านระบบ [[โครงสร้างดวงชะตา (格局)]] เสมอ
- "Affection" (情) ระหว่าง Ten Gods สำคัญกว่าจำนวน

**Modern Western:**
- มักนำ Ten Gods มาทำ personality profiling แบบ Jungian Archetypes
- Joey Yap นำเสนอ Ten Gods เป็น "character profiles" ที่เข้าใจง่าย

**ความขัดแย้ง:**
Seven Killings (七杀): สำนักหนึ่งมองว่าอันตรายต้องควบคุม อีกสำนักมองว่าเป็นพลังอำนาจที่มีค่า — ขึ้นอยู่กับ Pattern และ context ของดวง

## Common Misconceptions (ความเข้าใจผิด)

- ❌ "Ten Gods คือเทพเจ้าตามตัวอักษร" — เป็น Behavioral Patterns ไม่ใช่เทพ
- ❌ "ยิ่งมี Wealth Star เยอะยิ่งรวย" — ต้องดูว่า Day Master แข็งพอที่จะ "ถือ" ทรัพย์ได้ไหม
- ❌ "Seven Killings (七杀) เป็นลางร้ายเสมอ" — ในดวงที่ถูกต้อง 七杀 คือแหล่งอำนาจและความสำเร็จ
- ❌ "Hurting Officer (伤官) เป็นเรื่องแย่" — 伤官 ที่ใช้งานได้คือความฉลาดและความสามารถพิเศษ
