ระบบพยากรณ์เชิงสถานการณ์จีนโบราณที่ใช้ตารางกริด 3×3 (เก้าพระราชวัง 九宮) ซึ่งเปลี่ยนแปลงทุก 2 ชั่วโมงเพื่อแสดงพลังงานในขณะนั้น ประกอบด้วยเก้าดาว (九星) แปดประตู (八門) และแปดเทพ (八神) ที่กำหนดทิศทางและเวลาที่เหมาะสมสำหรับการตัดสินใจ QMDJ มีทั้งหมด 2,160 รูปแบบ (1,080 หยาง + 1,080 หยิน) และเป็นระบบที่มีความละเอียดเวลาสูงสุดในสามเหลี่ยม Chinese Metaphysics — [[Bazi (八字)]] อ่านรูปแบบชะตาชาติ / [[Zi Wei Dou Shu (紫微斗數)]] อ่าน event timing ระดับเดือน-ปี / QMDJ อ่านพลังงานสถานการณ์ทุก 2 ชั่วโมง

## ต้นกำเนิด

ตำนานระบุว่า QMDJ มาจากยุทธศาสตร์การรบของ Huangdi (黃帝 จักรพรรดิเหลือง) ที่ได้รับจาก Xuannu (玄女 เทพีลึกลับ) เพื่อเอาชนะ Chi You — ชื่อ "奇門遁甲" หมายถึง "ซ่อน Jia (甲 ก้านแรก) ใต้ก้านอื่นเพื่อหลีกเลี่ยง Geng (庚)" สะท้อนแนวคิดซ่อนพลังที่แท้จริงเพื่อความอยู่รอด Zhuge Liang (諸葛亮) ในยุค Three Kingdoms ถูกกล่าวขานว่าใช้ QMDJ ในการวางแผนการรบ หลังจากราชวงศ์หมิง QMDJ แยกออกจากดาราศาสตร์จริงและกลายเป็นระบบเชิงสัญลักษณ์ล้วน — กริดพระราชวังไม่ได้ map กับตำแหน่งดาวจริงอีกต่อไป

## โครงสร้างกริด — สองกระดาน

QMDJ ใช้กระดานซ้อนกัน 2 ชั้น:
- **Earth Board (地盤)** — กระดานดินที่ตรึงอยู่กับที่ แสดงตำแหน่งก้านดินหลัก
- **Heaven Board (天盤)** — กระดานฟ้าที่หมุนทุก 2 ชั่วโมง แสดงการเคลื่อนไหวของพลังงาน

## เก้าดาว (九星 Nine Stars)

| ดาว | ชื่อจีน | อิทธิพล |
|-----|---------|---------|
| Tianpeng (天蓬) | ดาวท้าวน้ำ | − อัปมงคล |
| Tianrui (天芮) | ดาวดิน | − อัปมงคล |
| Tianchong (天冲) | ดาวไม้ | + รุก/มงคลในการริเริ่ม |
| Tianfu (天輔) | ดาวไม้ช่วย | + มงคล |
| Tianqin (天禽) | ดาวดินกลาง | neutral ศูนย์กลาง |
| Tianxin (天心) | ดาวโลหะ | + มงคล |
| Tianzhu (天柱) | ดาวโลหะสอง | − อัปมงคล |
| Tianren (天任) | ดาวดินใหญ่ | + มงคล |
| Tianying (天英) | ดาวไฟ | neutral |

## แปดประตู (八門 Eight Gates)

| ประตู | ชื่อจีน | ระดับ |
|-------|---------|-------|
| Kai (開) | เปิด | ✓✓ ดีที่สุด |
| Xiu (休) | พัก | ✓ มงคล |
| Sheng (生) | เกิด | ✓✓ ดีที่สุด |
| Si (死) | ตาย | ✗✗ แย่ที่สุด |
| Jing (驚) | ตกใจ | ✗ อัปมงคล |
| Shang (傷) | บาดเจ็บ | ~ ไม่เป็นมิตร |
| Du (杜) | ปิดกั้น | ~ neutral-ลบ |
| Jing (景) | ทิวทัศน์ | ~ neutral |

*หมายเหตุ: 3 ประตูมงคล (Kai/Xiu/Sheng) + 2 อัปมงคล (Si/Jing) + 3 neutral*

## แปดเทพ (八神 Eight Gods)

Zhifu (值符) / Tengshe (腾蛇) / Taiyin (太陰) / Liuhe (六合) / Baihu (白虎) / Xuanwu (玄武) / Jiudi (九地) / Jiutian (九天) — แต่ละเทพแสดงบริบทสิ่งแวดล้อม บุคคล และพลังงานเสริมสำหรับการอ่านแต่ละพระราชวัง

## วิธีการอ่าน

- **Host/Guest (主客)** — ผู้อ่านระบุว่าตนเองเป็น Host (กำลังรับ) หรือ Guest (กำลังโจมตี/เริ่มต้น) เพื่อกำหนดว่าพระราชวังไหนเป็นพระราชวัง "ตนเอง"
- **Three-in-One Rule** — เมื่อประตู + ดาว + ก้านฟ้าตรงกันใน 3 ประตูมงคล (Kai/Xiu/Sheng) ในพระราชวังเดียวกัน = ผลลัพธ์ที่ดีที่สุด; ยิ่งมาก 3 ชั้นพร้อมกันยิ่งดี
- **Wu Xing Element Analysis** — ตรวจสอบปฏิสัมพันธ์ธาตุระหว่างประตู ดาว และเทพในพระราชวังเป้าหมาย เพื่อคาดการณ์ผลลัพธ์

## แนวคิดหลัก

- **เก้าพระราชวัง (九宮 Nine Palaces)** — กริด 3×3 เป็นพิกัดเชิงพื้นที่ พื้นฐานของระบบทั้งหมด
- **Yang Qi Men (陽遁)** — วิธีดั้งเดิมหลัก ใช้ 24 Solar Terms เป็นฐาน 540 รูปแบบหยาง
- **Yin Qi Men (陰遁)** — ปฏิทินสุริยะ-จันทรคติผสม 540 รูปแบบหยิน
- **Fa Qi Men (法奇門)** — กระแสเต๋า/ศาสนา ใช้ในทางจิตวิญญาณ
- **Fei Gong / Flying Palace** — วิธีที่ 4; ความถูกต้องถูกโต้แย้งโดยสำนักอื่น

## ข้อถกเถียงหลัก ⚠️

- **Destiny Reading vs Oracle QMDJ** — Joey Yap พัฒนาการอ่าน QMDJ จากวันเกิด (birth-based, natal chart) ซึ่งสำนักดั้งเดิมปฏิเสธ: QMDJ ดั้งเดิมเป็น event oracle (ใช้เวลาปัจจุบัน ไม่ใช่วันเกิด) เท่านั้น
- **สี่สำนัก** — Yang, Yin, Fa Qi Men, Fei Gong ไม่มีองค์กรกลางกำกับว่าสำนักใดถูกต้อง
- **ไม่มีการศึกษาเปรียบเทียบความแม่นยำ** — ไม่มีงานวิจัยเชิงประจักษ์ที่ใช้ตัวชี้วัดล่วงหน้าของ QMDJ กับผลลัพธ์จริงในกลุ่มตัวอย่างขนาดใหญ่

## สาเหตุที่สำคัญ

QMDJ เป็นเครื่องมือเสริมสำหรับการตัดสินใจทันที — ใช้คู่กับ [[Bazi (八字)]] (รูปแบบชะตาชาติ) และ [[Zi Wei Dou Shu (紫微斗數)]] (เหตุการณ์ระดับปี) เพื่อสร้างภาพรวมครบถ้วนในสามระดับเวลา
