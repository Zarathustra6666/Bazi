# Follow Patterns (从格)

> Pattern พิเศษที่ [[ธาตุเจ้าชะตา (Day Master)]] อ่อนมากจนต้อง **ยอมตาม** ธาตุที่ครอบงำดวง — แทนที่จะต่อต้าน [[ธาตุเจ้าชะตา (Day Master)|DM]] ยอมแพ้และกลายเป็นส่วนหนึ่งของกระแสนั้น ดวงที่ Follow Pattern แท้จริงเป็นหนึ่งในดวงที่ทรงพลังที่สุดในระบบ

---

## หลักการ

ต่างจาก Normal Patterns ที่ DM ใช้ Useful God เพื่อ "สมดุล" ตัวเอง — Follow Patterns DM **ละทิ้งตัวตนเดิม** และหนุนธาตุที่ครอบงำให้แข็งแกร่งขึ้น Useful God จึงคือธาตุที่เสริม dominant element ไม่ใช่ธาตุที่เสริม DM

**หายากมาก:** ประมาณ 1 ใน 1,000 ดวงเท่านั้นที่เป็น Follow Pattern แท้จริง

---

## เงื่อนไขร่วมทุก Follow Pattern

ต้องมีทั้งหมดนี้:

1. **DM ไม่มีราก (Rootless):** ไม่มี Branch ใดใน 4 เสาที่มี Hidden Stem ธาตุเดียวกับ DM
2. **ไม่มี Peer (比劫):** ไม่มี 比肩 หรือ 劫财 ในเสาใดเลย (Stem หรือ Branch Main Qi)
3. **Dominant element ครอบงำ:** ธาตุที่ DM ตามต้องมีอยู่ในสัดส่วนสูงในดวง

---

## ทั้ง 4 ประเภท

### 1. 从财格 (Cóng Cái Gé) — Follow Wealth

| เงื่อนไข | รายละเอียด |
|---------|-----------|
| DM | Rootless อย่างสมบูรณ์ |
| ไม่มี | Seal (印绶 8,9) — Seal ดูดพลัง Wealth กลับ |
| ไม่มี | Peers (比劫 0,1) — Peers แย่งชิง Wealth |
| ต้องมี | Wealth (偏财/正财) ครอบงำดวง — ใน Month Branch และ Stems |
| เสริม | Output (食伤) หนุน Wealth → chain สมบูรณ์ |

**ผลในชีวิต:** ความสามารถสร้างและคว้าโอกาสทางธุรกิจ ถ้าสมบูรณ์แบบ = ทรัพย์สินมหาศาล  
**ความเสี่ยง:** ถ้า LP/ปีนำ Seal หรือ Peers เข้า = Pattern พัง → ล้มละลาย/ปัญหาทางกฎหมาย

---

### 2. 从儿格 (Cóng Ér Gé) — Follow Output

| เงื่อนไข | รายละเอียด |
|---------|-----------|
| DM | Rootless อย่างสมบูรณ์ |
| ไม่มี | Seal (印绶 8,9) — Seal ทำลาย Output |
| ไม่มี | Peers (比劫 0,1) |
| ต้องมี | Output (食神/傷官) ครอบงำดวง — ใน Month Branch เป็นสำคัญ |
| เสริม | Wealth ที่ Output ผลิตออกมา → สมบูรณ์ |

**ผลในชีวิต:** ความฉลาดพิเศษ ความสามารถด้านศิลปะ/เทคนิค/นวัตกรรมสูงสุด  
**ความเสี่ยง:** Seal เข้า = พัง; Officer/Killing = ขัดแย้งและสูญเสียทรัพย์

---

### 3. 从杀格 (Cóng Shā Gé) — Follow Killings

| เงื่อนไข | รายละเอียด |
|---------|-----------|
| DM | Rootless อย่างสมบูรณ์ |
| ไม่มี | Output (食伤 2,3) — Output โจมตี Officer/Killing |
| ไม่มี | Peers (比劫 0,1) — Peers ต้านอำนาจ |
| ต้องมี | Officer/Killing (七杀/正官) ครอบงำดวง |
| เสริม | Wealth หนุน Officer/Killing |

**ผลในชีวิต:** ความสำเร็จในโครงสร้างอำนาจ (ทหาร รัฐบาล องค์กรใหญ่) ก้าวหน้าผ่านระบบลำดับชั้น  
**ความเสี่ยง:** Output เข้า = Pattern พัง → ความขัดแย้งกับผู้มีอำนาจ

---

### 4. 从势格 (Cóng Shì Gé) — Follow Momentum

| เงื่อนไข | รายละเอียด |
|---------|-----------|
| DM | Rootless อย่างสมบูรณ์ |
| ไม่มี | Seal (印绶) และ Peers (比劫) |
| ต้องมี | Wealth + Officer + Output ทั้งสามกลุ่มปรากฏพร้อมกัน |
| ลักษณะ | ไม่มีธาตุเดียวที่ครอบงำ แต่ DM ไหลตาม "กระแส" รวม |

**ผลในชีวิต:** ความยืดหยุ่นสูงสุด ปรับตัวได้ทุกสถานการณ์ หลายอาชีพและความสำเร็จหลายมิติ  
**ความเสี่ยง:** ต้องระวัง LP ที่นำ Seal หรือ Peers เข้า — เงื่อนไขเหมือนกัน

---

## Pure vs Flawed Follow Pattern

| ระดับ | เงื่อนไข | ผล |
|-------|---------|-----|
| **Pure (แท้จริง)** | DM ไม่มีรากเลย + dominant element สมบูรณ์ | ศักยภาพสูงสุด ความสำเร็จแน่นอนเมื่อ LP เสริม |
| **Flawed (มีรอยด่าง)** | DM มีรากเล็กน้อย หรือ Seal/Peer อ่อนมาก | ยังได้ผล แต่ต้องรอ LP ที่เอื้อ — มีความผันผวน |

---

## วิธีวิเคราะห์ Follow Pattern

1. **ตรวจ DM Rootlessness ก่อน** — ถ้ามีราก = ไม่ใช่ Follow Pattern
2. **ตรวจ Peer/Seal** — ถ้ามี = ไม่ใช่ Follow Pattern (หรือ Flawed)
3. **หา Dominant element** — TG กลุ่มใดครอบงำ (Wealth / Output / Officer)
4. **ตรวจ Month Branch** — Month Branch ต้องเป็น dominant element นั้น

**ตัวอย่างการตรวจใน App:**
- ตรวจว่า Branch hidden stems มีธาตุ DM หรือไม่ → rootless?
- นับ TG ใน stems + branch main qi → กลุ่มใด dominant?
- ตรวจ TG index 0-1 (Peer) และ 8-9 (Seal) → มีหรือไม่?

---

## ความสัมพันธ์กับ Pattern Purity

Follow Pattern แท้จริง = **[[Pattern Purity (格局純雜)|純格]] ระดับพิเศษ** เพราะดวงทั้งดวงไหลทิศทางเดียว แต่ใช้กฎ Useful God ต่างออกไป — ต้องเสริม dominant element ไม่ใช่เสริม DM

Follow Pattern Flawed = **有情格** — มีรูปแบบที่ดีแต่ไม่สมบูรณ์
