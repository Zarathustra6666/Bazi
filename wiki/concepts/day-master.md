# ธาตุเจ้าชะตา (日主)

> กิ่งฟ้าของเสาวัน — ตัวแทนของ "ตัวเรา" ในดวงชะตา และจุดอ้างอิงหลักของการตีความทั้งหมด

## Overview
ธาตุเจ้าชะตา (日主 Rì Zhǔ หรือ 日干 Rì Gān, Day Master) คือ [[กิ่งฟ้า (Heavenly Stems)]] ที่อยู่ใน [[สี่เสา (Four Pillars)|เสาวัน]] ซึ่งเป็นองค์ประกอบที่สำคัญที่สุดใน [[Bazi (八字)]] เพราะแทน "ตัวตน" ของเจ้าของดวง กิ่งฟ้าอื่นๆ ทั้งหมดในดวงชะตาจะถูกตีความผ่านความสัมพันธ์กับธาตุเจ้าชะตา — ความสัมพันธ์เหล่านั้นเรียกว่า [[สิบเทพ (十神)|Ten Gods]]

## Origin & History (ที่มา)
การกำหนดให้ Day Master (กิ่งฟ้าเสาวัน) เป็นจุดศูนย์กลางของดวงชะตาเป็นนวัตกรรมสำคัญที่สุดของ **Xu Ziping (徐子平)** แห่งราชวงศ์ซ่ง ก่อนหน้านั้น Li Xuzhong (ถัง) ใช้เสาปีเป็นจุดอ้างอิงหลัก การเปลี่ยนมาใช้เสาวันเป็นศูนย์กลางทำให้ Bazi เป็นศาสตร์ที่ **เฉพาะบุคคล** (individualized) อย่างแท้จริง และเป็นที่มาของคำว่า "Ziping Method (子平法)"

## Five Elements Relationship (ความสัมพันธ์ 五行)
Day Master สังกัด [[ธาตุทั้งห้า (Five Elements)]] หนึ่งตัว (และขั้ว หยาง/หยิน) — ความสัมพันธ์ระหว่าง Day Master กับ elements อื่นในดวงจะกำหนด [[สิบเทพ (十神)|Ten Gods]] ของแต่ละ element

- Elements ที่ **สร้าง DM** = Resource Stars (偏印, 正印)
- Elements ที่ **เหมือน DM** = Companion Stars (比肩, 劫财)
- Elements ที่ **DM สร้าง** = Output Stars (食神, 伤官)
- Elements ที่ **DM ควบคุม** = Wealth Stars (偏财, 正财)
- Elements ที่ **ควบคุม DM** = Authority Stars (七杀, 正官)

## Interaction Rules (กฎปฏิสัมพันธ์)

### การวิเคราะห์ความแข็งแกร่ง (强弱) — สองแนวทาง

**แนวทาง 1 — Balance-Based:**
- ประเมิน DM ว่าแข็งหรืออ่อนโดยดูจากจำนวน support elements
- **DM แข็ง (Strong):** ต้องการ Wealth, Authority, หรือ Output elements
- **DM อ่อน (Weak):** ต้องการ Resource หรือ Companion elements
- ฤดูกาลเกิด (Month Branch) มีผลต่อความแข็งแกร่งอย่างมาก

**แนวทาง 2 — Pattern-Based (Zi Ping Zhen Quan):**
- ไม่จำเป็นต้องตัดสิน "แข็ง/อ่อน" เป็นหลัก
- ดู [[โครงสร้างดวงชะตา (格局)|Pattern]] จาก Month Branch ก่อน
- [[เทพใช้งาน (用神)]] คือสิ่งที่ "fulfill" Pattern นั้น

### Special Structures (ดวงพิเศษ)
- **Follow Patterns (从格):** DM อ่อนมากจนต้อง "ยอมตาม" dominant element — ใน Follow Pattern DM อ่อนกลับเป็นดวงดีได้
- **Dominant Patterns (专旺格):** DM แข็งมากจนครองอำนาจ — ต้องการ elements เสริม ไม่ใช่ elements ต้าน
- ⚠️ ความเข้าใจผิดที่พบบ่อย: "ดวงแข็ง = ดวงดีเสมอ" — ผิด

## Interpretation Methods (วิธีตีความ)

**ขั้นตอนการอ่าน Day Master:**
1. ระบุ Day Master จากกิ่งฟ้าเสาวัน (1 ใน 10 ตัว)
2. ระบุฤดูกาลเกิดจาก Month Branch
3. ดูว่า Month Branch สนับสนุนหรือต้าน DM
4. ดู elements อื่นในดวงทั้งหมด
5. ตัดสิน Pattern หรือ Strength (ขึ้นอยู่กับสำนักที่ใช้)
6. หา [[เทพใช้งาน (用神)]]

**⚠️ หากตัดสิน Day Master ผิดตั้งแต่แรก → วิเคราะห์ผิดทุกอย่างที่ตามมา**

## Schools of Thought (สำนักต่างๆ)
ความขัดแย้งหลักในชุมชน Bazi เกี่ยวกับ Day Master คือ **"วิธีระบุ Useful God":**
- Balance School: ประเมิน DM strength → หา Useful God สมดุล
- Pattern School (ZPZQ): ดู Pattern จาก Month Branch → Useful God = element ที่ fulfill Pattern
- Climate School (Qiong Tong): ดูเดือนเกิด → Useful God = element ที่ปรับอุณหภูมิ

ดูรายละเอียดใน [[สำนักและวิธีตีความ Bazi]] และ [[เทพใช้งาน (用神)]]

## Common Misconceptions (ความเข้าใจผิด)
- ❌ "DM แข็ง = ดวงดีเสมอ" — Follow Patterns ที่ DM อ่อนมากก็เป็นดวงดีได้
- ❌ "ดูแค่ Day Master ก็รู้นิสัย" — Day Master เป็นแค่จุดเริ่มต้น ต้องดู Ten Gods ทั้งดวงประกอบ
- ❌ "มีสูตร % ที่แม่นยำในการตัดสิน strong/weak" — ต้องใช้ judgment ที่สั่งสมจากประสบการณ์
- ❌ "Day Master = สัตว์นักษัตร" — สัตว์นักษัตรคือก้านดินเสาปี ไม่ใช่ Day Master
