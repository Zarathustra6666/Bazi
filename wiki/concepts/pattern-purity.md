# Pattern Purity (格局純雜) — ความบริสุทธิ์ของโครงสร้างดวง

> หลักการจาก [[子平真诠 Zi Ping Zhen Quan]] ว่าด้วยความ "สะอาด" ของ [[โครงสร้างดวงชะตา (格局)|Pattern]] — ดวงที่ pure สร้างชีวิตที่ focused ดวงที่ mixed ต้องต่อสู้กับ internal friction

---

## 1. พื้นฐาน: 透出 (Transparent Stem) กับ Pattern

ก่อนเข้าใจ Pure/Mixed pattern ต้องเข้าใจกลไก **透出 (Tòu Chū — Transparent Stem)**

### 透出 คืออะไร

ใน Bazi แต่ละ Earthly Branch มี **Hidden Stems (藏干)** ซ่อนอยู่ภายใน เช่น:
- 子 (Zi, น้ำ) มี 癸 (Gui Water) ซ่อนอยู่
- 寅 (Yin, ไม้) มี 甲 (Jia Wood), 丙 (Bing Fire), 戊 (Wu Earth) ซ่อนอยู่
- 午 (Wu, ไฟ) มี 丁 (Ding Fire) เป็น Main Qi, 己 (Ji Earth) เป็น Hidden Qi

**透出 เกิดขึ้นเมื่อ:** Hidden Stem ที่ซ่อนอยู่ใน Branch ปรากฏขึ้นมาเป็น Heavenly Stem ในเสาใดเสาหนึ่ง (ปี เดือน หรือชั่วโมง)

**ผล:** Hidden Stem ที่ 透出 นั้นมีพลัง "พุ่งออกฟ้า — ซ่อนอยู่ในดิน" (天透地藏) ถือว่ามีอำนาจสูงสุด

### 透出 กำหนด Pattern อย่างไร

ตาม [[子平真诠 Zi Ping Zhen Quan]] ของ [[Shen Xiaozhan (沈孝瞻)]]:

1. ดูว่า Month Branch (เสาเดือน) มี Hidden Stems อะไรบ้าง
2. ถ้า Hidden Stem นั้น 透出 ขึ้นมาเป็น Heavenly Stem → นั่นคือ **Pattern ของดวง**
3. ถ้ามีหลาย Hidden Stems 透出 พร้อมกัน → ต้องใช้ Main Qi ของ Month Branch เป็นตัวกำหนด Pattern หลัก

**ตัวอย่าง:**
- Day Master คือ 甲 (Jia Wood) เกิดเดือน 酉 (You, Metal)
- 酉 มี 辛 (Xin Metal) เป็น Hidden Stem
- ถ้า 辛 ปรากฏเป็น Heavenly Stem ในเสาปี เดือน หรือชั่วโมง
- → 辛 Metal = Direct Officer (正官) ของ 甲 Wood
- → Pattern = **正官格 (Direct Officer Pattern)**
- → 透出 ทำให้ Pattern "ชัดเจน" และมีพลัง

## 2. สามระดับของ Pattern Purity

### ระดับที่ 1: 純格 (Pure Pattern)

**นิยาม:** Pattern ที่ชัดเจน สะอาด ไม่มีสิ่งรบกวน

**เงื่อนไขของ 純格:**
- Hidden Stem ของ Month Branch 透出 ขึ้นมา 1 ตัว (ไม่ใช่หลายตัวพร้อมกัน)
- Useful God (用神) ที่ support Pattern มีรากใน Earthly Branches
- ไม่มี Clash (冲), Penalty (刑), หรือ Combination ที่ทำลาย Useful God
- ไม่มี "Nemesis element" (element ที่ทำลาย Pattern) ใน Four Pillars

**คัมภีร์ ZPZQ กล่าวว่า:**
> 格局清純 → 貴 (Pattern บริสุทธิ์ → ความสูงส่ง)
> 格純則貴，格雜則賤 (Pattern บริสุทธิ์ → สูงศักดิ์, Pattern ปนเปื้อน → ต่ำต้อย)

**ผลในชีวิต:** ศักยภาพสูงสุด เส้นทางชีวิตชัดเจน ความสำเร็จค่อนข้างแน่นอนเมื่อ Luck Pillar เอื้อ

---

### ระดับที่ 2: 有情格 (Affectionate Pattern)

**นิยาม:** Pattern ที่ไม่สมบูรณ์แบบ แต่ Elements มี "ความสัมพันธ์ที่ดีต่อกัน" (情) — Useful God และ Pattern element ช่วยเหลือกัน แม้จะมีจุดอ่อน

**เงื่อนไขของ 有情格:**
- Pattern ชัดเจน แต่ Useful God อาจอ่อนเล็กน้อย หรือมีสิ่งรบกวนเล็กน้อยแต่ถูก "ควบคุม" โดย element อื่น
- Elements ในดวงมีความสัมพันธ์ที่ harmonic — ไม่มีการขัดแย้งรุนแรง
- Pattern "มีแรงหนุน" แม้จะไม่ 100% สะอาด

**ความต่างจาก 純格:**
| 純格 | 有情格 |
|------|--------|
| สมบูรณ์แบบ ไม่มีข้อบกพร่อง | ดีแต่ไม่สมบูรณ์ — มีจุดอ่อนที่ถูกชดเชย |
| ความสำเร็จค่อนข้างแน่นอน | ความสำเร็จมาเมื่อ Luck Pillar เอื้อ |
| หายาก | พบบ่อยกว่า |

**ผลในชีวิต:** ชีวิตที่ดีและมีความสำเร็จ แต่ต้องรอ Luck Pillar (大運) ที่เสริม Pattern ถึงจะถึงจุดพีค

---

### ระดับที่ 3: 雜格 (Mixed Pattern)

**นิยาม:** Pattern ที่ "ปนเปื้อน" — มีธาตุหรือ element ที่ขัดแย้งกับ Pattern หลักอยู่ด้วย

**สาเหตุของ 雜格:**
- มีหลาย Hidden Stems 透出 พร้อมกัน โดยไม่มีตัวใดชัดเจนเป็น main
- ตำราว่า "三柱帶格，雜難論" — สามเสาต่างมี Pattern ต่างกัน วุ่นวายยากวิเคราะห์
- Nemesis element ปรากฏใน Four Pillars โดยไม่ถูกควบคุม
- Useful God ถูก Clash หรือ Harm โดยตรง

**ผลในชีวิต:**
- ชีวิตไม่มีทิศทางชัดเจน หันมาหันไป ตัดสินใจยาก
- ต้องรอ Luck Pillar ที่ขจัด "Nemesis element" ออกก่อน ถึงจะเห็นความสำเร็จ
- ดวงที่ mixed มากอาจหมายถึงความผันผวนในชีวิตสูง

## 3. ตัวอย่างจาก ZPZQ

### ตัวอย่าง Pure Pattern: ดวงอัครมหาเสนาบดี Ye

```
เสาปี:  庚 (Geng) / 寅 (Yin)
เสาเดือน: 戊 (Wu) / 寅 (Yin)
เสาวัน:  庚 (Geng) / 寅 (Yin)
เสาชั่วโมง: 戊 (Wu) / 寅 (Yin)
```

- Heavenly Stems สลับกัน 庚-戊-庚-戊 (Geng Metal - Wu Earth) ครบ 4 เสา
- Earthly Branches เป็น 寅 (Yin, Wood) ทั้งหมด 4 เสา
- ความ pure ของ pattern นี้ถึงขนาดที่ตำราเรียกว่า "兩干不雜格 (Two Stem Pure Pattern)"
- ผล: ขึ้นสู่ตำแหน่งสูงสุดในรัฐบาล

### หลัก 透出 ที่ชัดเจน: 正官格 (Direct Officer Pattern)

```
Day Master: 甲 (Jia Wood)
Month Branch: 酉 (You Metal month)
Month Stem: 辛 (Xin Metal) → 透出 ขึ้นมา
```

- 辛 Metal ใน Month Stem คือ Direct Officer (正官) ของ 甲 Wood
- 透出 ทำให้ Pattern = **正官格** ชัดเจน
- Useful God = Seal (印绶) เพื่อ support Officer — ถ้ามี 印绶 อยู่ด้วยและไม่ถูก clash = 純格

### ตัวอย่าง Mixed Pattern

```
Day Master: 甲 (Jia Wood)
Month Branch: 寅 (Yin, ซึ่งมี 甲, 丙, 戊 ซ่อนอยู่)
```

ถ้า 丙 (Bing Fire, Eating God) 透出 ในเสาหนึ่ง AND 戊 (Wu Earth, Direct Wealth) 透出 ในอีกเสาหนึ่ง:
- ดวงนี้มีทั้ง Eating God Pattern และ Wealth Pattern
- เรียกว่า "雜格" — วิเคราะห์ยาก ต้องดูว่า Pattern ไหน "stronger"

## 4. กฎการตัดสิน Pure vs Mixed

### กฎจาก ZPZQ

1. **กฎ Main Qi:** ถ้า Main Qi ของ Month Branch 透出 = Pattern ชัดเจน
2. **กฎ One God:** ถ้ามี Useful God เพียง 1 ตัวที่ชัดเจน = ดีกว่ามีหลายตัว
3. **กฎ No Mixing:** ถ้า Two Stems ไม่ 透出 พร้อมกัน = 清 (Pure)
4. **กฎ Seasonal Strength:** Hidden Stem ที่มีฤดูกาลสนับสนุน = แข็งแกร่ง มีอำนาจ

### สัญญาณของ Pure Pattern
- เสาเดือน Stem ตรงกับ Hidden Stem ของ Month Branch
- ไม่มี element ที่ clash หรือ harm Useful God
- Useful God มีรากใน Earthly Branches (ไม่ "Rootless")
- ประโยคคลาสสิก: "清而且純" = สะอาดและบริสุทธิ์

### สัญญาณของ Mixed Pattern
- หลาย Ten Gods 透出 พร้อมกันจาก Month Branch
- มี Clash ที่ทำลาย Useful God โดยตรง
- Pattern หลักและ Pattern รอง "ขัดแย้งกัน" (ต้องการ Useful God ที่ต่างกัน)

## 5. ความสำคัญของ 透出 ต่อ Pattern Purity — สรุป

| สถานการณ์ | ผลต่อ Pattern |
|-----------|--------------|
| Hidden Stem 透出 ใน Month Stem | Pattern แข็งแกร่งที่สุด (天透地藏) |
| Hidden Stem 透出 ใน Year หรือ Hour Stem | Pattern มีอยู่แต่อ่อนกว่า |
| Hidden Stem ไม่ 透出 เลย | Pattern ยังอยู่ใน Earthly Branches แต่อำนาจต่ำ |
| Hidden Stem หลายตัว 透出 พร้อมกัน | 雜格 — Pattern ปนเปื้อน |
| Hidden Stem 透出 แต่ถูก Clash ในดวง | Pattern ถูก damage — อ่อนแอ |

## Open Questions

- ถ้า Month Branch มี Main Qi ไม่ 透出 แต่ Minor Hidden Stem 透出 — ควรใช้ Minor Stem เป็น Pattern หรือไม่? (ZPZQ ใช้ Main Qi, แต่บางสำนักใช้ Stem ที่ transparent)
- ขอบเขตระหว่าง 有情格 และ 雜格 ยังไม่มีคำตอบชัดเจน — เป็น judgment call ของผู้อ่านดวง
