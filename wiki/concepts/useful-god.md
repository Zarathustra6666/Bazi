# เทพใช้งาน (用神)

> Element หลักที่ทำหน้าที่เป็น "ยา" สำหรับความไม่สมดุลในดวงชะตา — "จักรพรรดิ" แห่งดวง

## Overview
用神 (Yòng Shén, Useful God หรือ Favorable God) คือ element ที่สำคัญที่สุดในดวงชะตาหนึ่งๆ ทำหน้าที่รักษาสมดุลหรือ "fulfill" โครงสร้างของดวง ถูกเปรียบว่าเป็น **"จักรพรรดิ"** แห่งดวงชะตา การรู้จัก Useful God คือกุญแจสู่การตีความ [[Bazi (八字)]] ที่ถูกต้อง — [[วงจรดวงชะตา (大運)|Luck Pillars]] ที่เสริม Useful God คือช่วงเวลาดีในชีวิต

## Origin & History (ที่มา)
แนวคิด用神ปรากฏในคัมภีร์ [[渊海子平 Yuan Hai Zi Ping]] (ซ่ง) และได้รับการพัฒนาอย่างละเอียดใน [[子平真诠 Zi Ping Zhen Quan]] โดย Shen Xiaozhan (ชิง) ซึ่งผูก Useful God เข้ากับระบบ [[โครงสร้างดวงชะตา (格局)]] ขณะที่ [[穷通宝鑑 Qiong Tong Bao Jian]] นำเสนอแนวทาง 调候用神 (Climate Adjustment Useful God) ที่ต่างออกไป ปัจจุบันสองแนวทางนี้ยังคงเป็นประเด็นถกเถียงหลักในชุมชน Bazi ทั่วโลก

## Core Definition (นิยามหลัก)
用神 = element ที่:
- **รักษาความสมดุล** ของดวงชะตา (Balance approach) หรือ
- **fulfill** (สนับสนุนให้สำเร็จ) [[โครงสร้างดวงชะตา (格局)|Pattern]] หลักของดวง (Pattern approach)

**ความสำคัญ:**
- หาก Useful God **แข็งแกร่งและไม่ถูกรบกวน** → ดวงดี ชีวิตราบรื่น
- หาก Useful God **อ่อนแอหรือถูกโจมตี** → ดวงตกต่ำ ต้องสู้กับอุปสรรค
- **忌神 (Jì Shén, Unfavorable God)** = element ที่โจมตี Useful God

## Five Elements Relationship (ความสัมพันธ์ 五行)
Useful God มักเป็น element ที่:
- สร้างสมดุลให้ดวง (Balance approach)
- สนับสนุนหรือ protect Pattern หลัก (Pattern approach)
- "ปรับอุณหภูมิ" ให้ Day Master เจริญได้ตามฤดูกาล (Climate approach)

**Month Branch** คือแหล่งที่มาหลักของ Useful God ในทั้งสามแนวทาง

## Interaction Rules (กฎปฏิสัมพันธ์)
- Useful God ต้องมี **Qi** (พลังงาน) เพียงพอ — ไม่ใช่แค่ปรากฏในดวง
- Useful God ที่ถูก **clash หรือ combine** จะทำงานได้ไม่เต็มที่
- Useful God ที่ "transparent" (透出 ขึ้นมาใน Heavenly Stems) = แข็งแกร่งขึ้น
- [[วงจรดวงชะตา (大運)|Luck Pillars]] ที่นำ Useful God มาเสริม = ช่วงเวลาที่ดีที่สุดในชีวิต

## Interpretation Methods (วิธีตีความ)

### แนวทางที่ 1 — Balance-Based
```
1. ประเมิน Day Master ว่าแข็งหรืออ่อน
2. หา element ที่สร้างสมดุล:
   - DM แข็ง → Useful God = สิ่งที่ควบคุมหรือดูดซับพลัง DM
   - DM อ่อน → Useful God = สิ่งที่สนับสนุนหรือสร้าง DM
```

### แนวทางที่ 2 — Pattern-Based (Zi Ping Zhen Quan)
```
1. ระบุ格局 (Pattern) จาก Month Branch ก่อน
2. Useful God = element ที่ "fulfill" Pattern นั้น
   เช่น Direct Officer Pattern → element ที่ปกป้อง Officer = Useful God
3. ความแข็ง/อ่อนของ Day Master เป็นปัจจัยรอง
```

### แนวทางที่ 3 — Climate Adjustment (Qiong Tong Bao Jian)
```
Useful God = element ที่ "ปรับอุณหภูมิ" ดวงให้สมดุลตามฤดูกาล
- เกิดเดือนหนาว (亥子丑) → ต้องการไฟ
- เกิดเดือนร้อน (巳午未) → ต้องการน้ำ
```

## Schools of Thought (สำนักต่างๆ)

**ความขัดแย้งใหญ่ที่สุดในชุมชน Bazi สมัยใหม่:**

| สำนัก | แนวทาง Useful God | คัมภีร์อ้างอิง |
|-------|-----------------|--------------|
| Pattern School | Pattern-based | [[子平真诠 Zi Ping Zhen Quan]] |
| Balance School | Strength-based | หลายคัมภีร์ |
| Climate School | Season/Month-based | [[穷通宝鑑 Qiong Tong Bao Jian]] |

ดวงชะตาเดียวกัน 2 สำนักให้ Useful God ต่างกัน → การวิเคราะห์ Luck Pillars ที่ตามมาจะต่างกันทั้งหมด

### The ZPZQ vs QTBJ Contradiction

The two most influential classical texts give directly contradictory Useful God prescriptions for certain chart configurations:

**Example: Winter Jia Wood (甲木) Day Master, Month Branch 子 (Zi/Rat, Water)**
- **ZPZQ method**: Month Branch is Zi Water → pattern flows toward Water → Yong Shen is whatever supports the Water pattern (e.g., Metal producing Water, or Water itself if strong enough)
- **QTBJ method**: Winter chart is cold/frozen → Day Master needs warmth → "medicine is fire" → Yong Shen is Bing Fire (丙火) to thaw the chart, *even if this destroys the Water pattern*

There is no agreed meta-rule in classical literature for which school takes precedence. Practitioners blending both systems will routinely produce different Yong Shen — and therefore different life advice — for identical charts. This is the central interpretive problem of classical Bazi.

See [[Comparative Analysis — Classical Texts, Schools, and Regional Traditions|comparative-sources]] for full cross-text analysis.

## A4: Useful God as Life Direction — แนวทางชีวิตตาม Useful God

ความรู้จาก Useful God ไม่ใช่แค่ "ธาตุที่ดี" — มันบอกว่าควร **focus ชีวิตในทิศทางไหน** เพื่อให้ดวงทำงานได้เต็มที่

### Useful God ต่อธาตุ — แนวทางชีวิตจริง

| Useful God | ธาตุ | แนวทางอาชีพ | แนวทางความสัมพันธ์ | แนวทางการเงิน |
|-----------|------|-----------|-----------------|------------|
| **ไม้ (Wood)** | การเติบโต เมตตา การศึกษา | สุขภาพ สิ่งแวดล้อม การศึกษา ที่ปรึกษา | ความสัมพันธ์ที่เน้นการเติบโตร่วมกัน | ลงทุนระยะยาว ธุรกิจที่เติบโตช้าแต่มั่นคง |
| **ไฟ (Fire)** | ความกล้า การสื่อสาร เครือข่าย | สื่อ บันเทิง PR นักพูด งาน social | ความสัมพันธ์ที่มีชีวิตชีวา ชอบกิจกรรมสังคม | รายได้จากการสร้าง brand และเครือข่าย |
| **ดิน (Earth)** | ความมั่นคง ความน่าเชื่อถือ การออม | อสังหาริมทรัพย์ การเกษตร การเงิน บัญชี | ความสัมพันธ์มั่นคง ครอบครัวเป็นหลัก | ออมและลงทุนในสินทรัพย์ถาวร ที่ดิน |
| **โลหะ (Metal)** | ความยุติธรรม ระเบียบ การตัดสินใจ | กฎหมาย การเงิน วิศวกรรม การแพทย์ | ความสัมพันธ์ที่มีหลักการ ซื่อสัตย์ | วางแผนการเงินรัดกุม ลงทุนในระบบ |
| **น้ำ (Water)** | ปัญญา ความยืดหยุ่น การเดินทาง | วิจัย เทคโนโลยี ปรัชญา การท่องเที่ยว | ความสัมพันธ์ที่มีความลึก ต้องการพื้นที่ | ลงทุนในความรู้และทักษะ income จากปัญญา |

### เมื่อ Luck Pillar เสริม Useful God — สัญญาณที่ควรทำ

```
UG = ไม้: ช่วง LP Wood → เรียนเพิ่ม เปิด startup ด้านสุขภาพ ช่วยเหลือผู้อื่น
UG = ไฟ: ช่วง LP Fire → ขยายเครือข่าย เริ่มธุรกิจ social สร้าง personal brand
UG = ดิน: ช่วง LP Earth → ซื้ออสังหาฯ ลงทุนในสิ่งที่จับต้องได้ สร้างความมั่นคง
UG = โลหะ: ช่วง LP Metal → พัฒนาทักษะ ขอเลื่อนตำแหน่ง ลงทุนตามระบบ
UG = น้ำ: ช่วง LP Water → เรียนรู้สิ่งใหม่ เดินทาง ทำวิจัย พัฒนา IT
```

### เมื่อ Luck Pillar เป็น 忌神 (Unfavorable) — สัญญาณที่ควรระวัง

```
UG ถูกโจมตี → อย่าตัดสินใจใหญ่ในช่วงนี้
LP เป็นธาตุที่ควบคุม UG → ความท้าทาย — ต้องรักษา UG ให้แข็งแกร่ง
LP นำ Nemesis ของ Pattern เข้า → วิกฤต — ระมัดระวังสูงสุด
```

**หมายเหตุ:** แนวทางนี้เป็นการประยุกต์ใช้ทั้ง Balance-based และ Pattern-based approach — ไม่ใช่กฎตายตัวในคัมภีร์คลาสสิก แต่เป็น synthesis ที่ใช้งานได้ในทางปฏิบัติ

## Common Misconceptions (ความเข้าใจผิด)

- ❌ "Useful God มีตัวเดียวเสมอ" — บางดวงมีหลาย Useful Gods ที่ทำงานร่วมกัน
- ❌ "วิธีคำนวณ % Strength แม่นยำ" — ไม่มีสูตรตายตัว ต้องใช้ judgment ที่สั่งสมจากประสบการณ์
- ❌ "สองแนวทางให้ผลเหมือนกัน" — Balance-based และ Pattern-based มักให้ Useful God ต่างกันในดวงเดียวกัน
- ❌ "Useful God = ธาตุที่ควรสวมใส่หรือล้อมรอบตัว" — เป็นการตีความแบบ pop culture ไม่ใช่ classical Bazi
