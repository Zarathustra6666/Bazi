# 格局 Pattern Life Narratives — เส้นทางชีวิตตาม Pattern

> แต่ละ [[โครงสร้างดวงชะตา (格局)|格局 Pattern]] ไม่ได้แค่บอกโครงสร้างดวงชะตา — มันกำหนด **เส้นทางชีวิต วิถีอาชีพ รูปแบบทรัพย์สิน และความท้าทายที่ต้องเผชิญ** ตาม [[子平真诠 Zi Ping Zhen Quan]] ของ Shen Xiaozhan

---

## 1. หลักการอ่าน Pattern Life Narrative

### Pattern กำหนดอะไร

จาก [[子平真诠 Zi Ping Zhen Quan]] (ZPZQ): Pattern ถูกกำหนดจาก **Main Qi ของ Month Branch** → ได้ Ten God ของ Month Branch → นั่นคือ Pattern ของดวงชะตา

Pattern บอก:
- **เส้นทางธรรมชาติ** (Natural path) — ชีวิตจะไหลไปในทิศทางนี้โดยอัตโนมัติ
- **[[เทพใช้งาน (用神)|Useful God]]** — ธาตุ/Ten God ที่ต้องส่งเสริมให้ Pattern ทำงานได้ดี
- **ศัตรูของ Pattern** (Nemesis) — สิ่งที่ทำลาย Pattern และก่อปัญหา
- **เงื่อนไขความสำเร็จ** — ต้องใช้ Luck Pillar ช่วง Useful God เข้า

### ความบริสุทธิ์ของ Pattern (純格 vs 雜格)

| ระดับ | ลักษณะ | ผลลัพธ์ |
|-------|--------|---------|
| **純格 (Pure)** | Useful God แข็งแกร่ง ไม่มี Nemesis element ในดวง | ศักยภาพสูงสุด ความสำเร็จค่อนข้างแน่นอน |
| **有情格 (Affectionate)** | Pattern ชัดเจน มี Useful God แต่ไม่สมบูรณ์แบบ | ความสำเร็จมาเมื่อ Luck Pillar เอื้อ |
| **雜格 (Impure)** | มีธาตุที่ทำลาย Pattern อยู่ด้วย | ต้องรอ Luck Pillar ขจัด Nemesis ออก จึงจะสำเร็จ |

**กฎหลัก (ZPZQ):** Pattern บริสุทธิ์และ Luck Pillar เสริม Pattern → ชีวิตรุ่งเรือง / Pattern ขัดแย้งกัน Luck Pillar ทำลาย Pattern → ชีวิตวุ่นวาย

## 2. แปด Normal Patterns (正格)

### 2.1 建禄格 — Commander Pattern

> **สาระ:** [[ธาตุเจ้าชะตา (Day Master)|Day Master]] แข็งแกร่งสมบูรณ์ — เสาเดือนเป็น "Maturity" (临官) ของ DM

**เส้นทางชีวิต:**
บุคคลเกิดมาพร้อมพลังงานของตัวเอง ไม่ต้องพึ่งพาสิ่งอื่น → เส้นทางของผู้สร้างตัวเองจากศูนย์ คลาสสิกระบุว่า建禄格 "ดึงดูดสิ่งลึกลับและจิตวิญญาณ" มักมีความคิดพุ่งออกนอกกระแส

**อาชีพที่เหมาะ:**
ธุรกิจอิสระ, เป็นเจ้าของกิจการ, งานที่ต้องใช้ความสามารถส่วนตัวล้วนๆ — ไม่ชอบงานในระบบองค์กรใหญ่

**ทรัพย์สิน:**
ตำราคลาสสิก: *"建禄不富"* — Commander Pattern มักไม่สะสมทรัพย์ได้มาก เพราะ Month Branch "เสาเดือนเป็นของ DM เอง" → Peers แย่งส่วนแบ่ง ต้องมี Wealth หรือ Officer ใน Hour/Year pillar เพื่อชดเชย

**Useful God:** ทรัพย์ (Wealth stars) หรือ Officer stars — ช่วยให้พลังงานส่วนตัวมีทิศทาง
**Nemesis:** Peers/Rob Wealth ที่มากเกินไป — แย่งทรัพย์, ขัดแย้ง

**เมื่อ Luck Pillar เข้า Wealth element:** จุดพีคด้านการเงิน — ธุรกิจเติบโต
**เมื่อ Luck Pillar เข้า Officer element:** ชื่อเสียงและตำแหน่ง — แต่ต้องไม่ขัดแย้งกับ DM

### 2.2 月刃格 — Yang Blade Pattern

> **สาระ:** DM หยางเกิดในเดือน "Peak" (帝旺) — พลังงานล้นเกิน

**เส้นทางชีวิต:**
พลังงานสูงมาก ก้าวร้าว กล้าหาญ ชอบความท้าทาย — คล้าย七杀格 แต่มาจากพลังงาน DM เอง ไม่ใช่ external pressure

**อาชีพที่เหมาะ:**
ทหาร, ศัลยแพทย์, กีฬา, ผู้บริหารระดับสูง — อาชีพที่ต้องการความกล้าและความแข็งแกร่ง

**ทรัพย์สิน:**
มีโอกาสรวยถ้า 七杀 (Seven Killings) ควบคุม Blade ได้ — "Blade uniting with Killings" (阳刃合杀) = ดีมาก

**Useful God:** Seven Killings หรือ Direct Officer — ควบคุมพลังงานที่ล้นเกิน
**Nemesis:** ไม่มีสิ่งควบคุม → ชีวิตวุ่นวาย อุบัติเหตุ

**เมื่อ Luck Pillar เข้า Officer/Killings:** จุดพีคแห่งความสำเร็จ — พลังงานถูกชี้นำ
**เมื่อ Luck Pillar เข้า Resource:** เสริมพลัง แต่ระวังแข็งเกินไป

---

### 2.3 食神格 — Eating God (Talent) Pattern

> **สาระ:** Month Branch Hidden Stem = Output ขั้ว Same (食神) — ความสุขและพรสวรรค์

**เส้นทางชีวิต:**
ชีวิตไหลเรื่อยๆ ด้วยพรสวรรค์ธรรมชาติ — ไม่ต้องพยายามมากทรัพย์ก็มา, สุขภาพดี, มีเสน่ห์ที่น่ารักใคร่ คนรอบข้างชอบ คลาสสิกเรียกว่า "ดาวแห่งอายุยืนและทรัพย์"

**อาชีพที่เหมาะ:**
ศิลปิน, นักเขียน, ธุรกิจอาหาร/การท่องเที่ยว, บันเทิง, การศึกษา — งานที่สร้างสรรค์และให้ความสุขแก่ผู้อื่น

**ทรัพย์สิน:**
食神生财 — Eating God produces Wealth naturally → ทรัพย์มาเองโดยไม่ต้องพยายามมาก Pure食神格 เป็น Pattern ที่ "สะสมทรัพย์ได้ดี" ตามคลาสสิก

**Useful God:** Indirect Wealth (偏财) — Eating God → Wealth chain ที่สมบูรณ์
**Nemesis:** Indirect Seal (枭神/偏印) — "Seal steals Eating God's food" (枭神夺食) = โชคร้าย ทรัพย์หาย สุขภาพแย่

**เมื่อ Luck Pillar เข้า Wealth element:** ทรัพย์รุ่งเรือง — ดีที่สุดสำหรับ Pattern นี้
**เมื่อ Luck Pillar เข้า Indirect Seal:** ช่วงที่ยากที่สุด — ต้องระวังสุขภาพและการสูญเสีย

---

### 2.4 傷官格 — Hurting Officer Pattern

> **สาระ:** Month Branch = Output ขั้ว Opposite (傷官) — ความคิดสร้างสรรค์ที่เฉียบคม

**เส้นทางชีวิต:**
ฉลาดสูงมาก คิดนอกกรอบ กล้าท้าทายระบบ — แต่ชีวิตมักมีความขัดแย้งกับผู้มีอำนาจ ตำราคลาสสิกบอกว่า傷官格 มักผ่านความยากลำบาก แต่สร้างสรรค์สิ่งยิ่งใหญ่

**อาชีพที่เหมาะ:**
ศิลปิน, นักดนตรี, ทนายความ (ชอบโต้เถียง), นักเขียน, นักวิทยาศาสตร์, ผู้ประกอบการที่ไม่ทำตามกฎ

**ทรัพย์สิน:**
傷官生财 — ทรัพย์มาจากความสามารถพิเศษ แต่ไม่เท่าเทียม มีทั้งดีและแย่สลับกัน

**Useful God:**
- **傷官配印**: ถ้า DM อ่อน → ต้องการ Resource (印) ช่วยเสริม → งานวิชาการ/ศิลปะที่ได้รับการยอมรับ
- **傷官生财**: ถ้า DM แข็ง → Hurting Officer → Wealth chain → ธุรกิจ
- **傷官驾杀**: ถ้ามี Seven Killings → ให้ Hurting Officer ควบคุม → ผู้นำที่กล้า
**Nemesis:** Direct Officer (正官) — *"傷官见官，为祸百端"* (Hurting Officer meets Officer = ภัยพิบัติร้อยประการ) — อย่ารับตำแหน่งราชการโดยตรง

**เมื่อ Luck Pillar เข้า Wealth element:** ทรัพย์จากพรสวรรค์ — จุดพีค
**เมื่อ Luck Pillar เข้า Direct Officer:** ช่วงที่ยากที่สุด — ความขัดแย้งสูงสุด

### 2.5 偏财格 — Indirect Wealth (Venturer) Pattern

> **สาระ:** Month Branch = Indirect Wealth (偏财) — ทรัพย์จากโชคและโอกาส

**เส้นทางชีวิต:**
เกิดมาพร้อมพรสวรรค์ด้านการหาเงิน เข้าสังคมเก่ง มีเสน่ห์ดึงดูดโอกาส — ทรัพย์ไหลเข้ามาจากหลายทาง แต่ใช้จ่ายเร็วเช่นกัน ชีวิตมีสีสันและความตื่นเต้น

**อาชีพที่เหมาะ:**
นักลงทุน, ธุรกิจ, การขาย, บันเทิง, การค้าระหว่างประเทศ — งานที่ต้องเจรจาและสร้างความสัมพันธ์

**ทรัพย์สิน:**
สะสมได้เร็วกว่า Direct Wealth แต่ไม่มั่นคงเท่า ตำราคลาสสิกบอกว่า "เงินมาเงินไป" — ต้องมี Output stars เพื่อสร้างทรัพย์ต่อเนื่อง

**Useful God:** Output stars (食神/傷官) เพื่อ generate ทรัพย์; หรือ Resource เพื่อรักษาทรัพย์
**Nemesis:** Peers/Rob Wealth (比劫) — "พี่น้องแย่งเงิน" — ทรัพย์สูญหายผ่านคนใกล้ชิด

**เมื่อ Luck Pillar เข้า Output element:** โอกาสธุรกิจใหม่ — ทรัพย์เพิ่มขึ้น
**เมื่อ Luck Pillar เข้า Peers/Rob Wealth:** ระวังการสูญเสียทรัพย์ผ่านผู้อื่น

---

### 2.6 正财格 — Direct Wealth Pattern

> **สาระ:** Month Branch = Direct Wealth (正财) — ทรัพย์จากความพยายามและระเบียบวินัย

**เส้นทางชีวิต:**
ชีวิตมีระเบียบ รับผิดชอบ น่าเชื่อถือ — ทรัพย์สะสมช้าแต่มั่นคง เป็นคนที่คนรอบข้างไว้วางใจ ไม่เสี่ยง ชอบความมั่นคง

**อาชีพที่เหมาะ:**
ราชการ, การธนาคาร, บัญชี, อสังหาริมทรัพย์, ธุรกิจที่มั่นคง — งานที่ได้เงินเดือนประจำหรือรายได้ที่คาดการณ์ได้

**ทรัพย์สิน:**
สะสมได้มั่นคง "ยึดเงินแน่น" — มีที่ดินและสินทรัพย์ถาวร ต้องมี Output stars หรือ Officer เพื่อปกป้องทรัพย์จาก Peers

**Useful God:** Direct Officer (正官) ปกป้องทรัพย์จากการโจมตีของ Peers; หรือ Output stars สร้างทรัพย์เพิ่ม
**Nemesis:** Peers/Rob Wealth — โจมตีทรัพย์โดยตรง

**เมื่อ Luck Pillar เข้า Officer element:** ชื่อเสียงและทรัพย์ไปด้วยกัน — จุดสูงสุดในอาชีพ
**เมื่อ Luck Pillar เข้า Rob Wealth:** ระวังการสูญเสีย ความขัดแย้งทางธุรกิจ

### 2.7 七杀格 — Seven Killings (Warlord) Pattern

> **สาระ:** Month Branch = Seven Killings (七杀) — แรงกดดันกลายเป็นพลัง

**เส้นทางชีวิต:**
ชีวิตเต็มไปด้วยความท้าทายและแรงกดดัน → แต่คนที่ผ่านได้จะกลายเป็นผู้นำที่แข็งแกร่งมาก — คลาสสิกบอกว่า七杀格 ที่ได้รับการควบคุมอย่างถูกต้อง = ดวงของนายพล/ผู้บริหารระดับสูง

**อาชีพที่เหมาะ:**
ทหาร/ตำรวจ, ศัลยแพทย์, CEO ในอุตสาหกรรมที่มีการแข่งขันสูง, นักการเมือง, นักกีฬาอาชีพ

**ทรัพย์สิน:**
ทรัพย์มาพร้อมความเสี่ยงสูง — "ความร่ำรวยผ่านสงคราม" แต่ถ้า Pattern แข็งแกร่งและมีการควบคุม = ทรัพย์มหาศาล

**Useful God สามแบบ:**
- **食神制杀 (Eating God tames Killings):** สร้างพลังเชิงบวก → ความสำเร็จผ่านพรสวรรค์
- **印绶化杀 (Seal transforms Killings):** ความรู้และปัญญาเปลี่ยนความรุนแรงเป็นพลัง → ผู้นำที่มีวิสัยทัศน์
- **阳刃合杀 (Yang Blade unites with Killings):** พลังงานแข็งกร้าวรวมกัน → แม่ทัพ/นักรบ
**Nemesis:** Pattern ที่ไม่มีการควบคุม (无制化) = ชีวิตวุ่นวาย อันตราย สูญเสีย

**เมื่อ Luck Pillar เข้า Useful God element (印/食/刃):** จุดพีคแห่งอำนาจ
**เมื่อ Luck Pillar เข้า Wealth:** เสริม Killings → ระวัง ถ้า Killings แข็งอยู่แล้วจะยิ่งวุ่นวาย

### 2.8 正官格 — Direct Officer (Executive) Pattern

> **สาระ:** Month Branch = Direct Officer (正官) — ระเบียบ อำนาจ และความรับผิดชอบ

**เส้นทางชีวิต:**
ชีวิตเดินตามกฎและระเบียบ มีเกียรติในสังคม ได้รับความไว้วางใจ — เส้นทางของข้าราชการ นักบริหาร ผู้นำองค์กร ตำราคลาสสิกบอกว่านี่คือ "Pattern ของขุนนาง"

**อาชีพที่เหมาะ:**
ราชการ, ผู้พิพากษา, ผู้บริหารองค์กร, นักการทูต, อาจารย์มหาวิทยาลัย, กฎหมาย

**ทรัพย์สิน:**
ทรัพย์มาจากตำแหน่งและสถานะ — ไม่รวยเร็ว แต่มั่นคงและยาวนาน มีเกียรติและชื่อเสียงไปพร้อมกัน

**Useful God:** Resource (印绶) — ปกป้อง Officer จาก Hurting Officer; Direct Wealth (正财) เสริมให้ Officer ทำงานได้ดี
**Nemesis:** Hurting Officer (傷官) — ทำลาย Officer โดยตรง; Seven Killings ผสมกัน (官杀混杂) — ทำให้อำนาจสับสน

**เมื่อ Luck Pillar เข้า Resource element:** การศึกษาและตำแหน่งเจริญ — จุดสูงสุด
**เมื่อ Luck Pillar เข้า Hurting Officer:** ความขัดแย้งกับผู้มีอำนาจ — ระวังการสูญเสียตำแหน่ง

---

### 2.9 偏印格 — Indirect Seal (Mystic Resource) Pattern

> **สาระ:** Month Branch = Indirect Seal (偏印) — ความรู้นอกกระแสและสัญชาตญาณพิเศษ

**เส้นทางชีวิต:**
ฉลาดแต่แตกต่าง — ความรู้ที่สะสมมาจาก "ทางด้านข้าง" ไม่ใช่ระบบปกติ มีสัญชาตญาณแหลมคม บางครั้งโดดเดี่ยว แต่เป็นผู้เชี่ยวชาญในสาขาเฉพาะ

**อาชีพที่เหมาะ:**
นักวิจัย, ผู้เชี่ยวชาญเฉพาะทาง, ศิลปะลึกลับ, เทคโนโลยีใหม่, ปรัชญา, โหราศาสตร์

**ทรัพย์สิน:**
ทรัพย์จากทักษะพิเศษและงานเฉพาะทาง — ไม่มากแต่มีคุณภาพ ต้องการ Output stars เพื่อ "ขาย" ความรู้

**Useful God:** Output stars (食神/傷官) — เปลี่ยนความรู้เป็นทรัพย์
**Nemesis:** Eating God ถูก Indirect Seal "ขโมย" (枭神夺食) — ระวัง Pattern นี้ทำลาย食神格

**เมื่อ Luck Pillar เข้า Output element:** ความสำเร็จจากทักษะพิเศษ
**เมื่อ Luck Pillar เข้า Wealth:** ทรัพย์จากความเชี่ยวชาญ

---

### 2.10 正印格 — Direct Seal (Resource) Pattern

> **สาระ:** Month Branch = Direct Seal (正印) — การสนับสนุน การศึกษา และปัญญา

**เส้นทางชีวิต:**
ได้รับการสนับสนุนจากผู้อื่น (พ่อแม่ ครู ผู้อุปถัมภ์) ชีวิตมีพื้นฐานมั่นคง การศึกษาดี เคารพกฎและประเพณี — เส้นทางของนักวิชาการและครู

**อาชีพที่เหมาะ:**
อาจารย์, นักวิชาการ, นักเขียน, แพทย์, นักกฎหมาย, ที่ปรึกษา — งานที่ใช้ความรู้และได้รับความเคารพ

**ทรัพย์สิน:**
ทรัพย์จากการศึกษาและตำแหน่ง ไม่ใช่การค้า — มีที่ดินและสินทรัพย์ถาวร Wealth มากเกินไปจะทำลาย Resource (财破印) = เสียโอกาสทางการศึกษา

**Useful God:** Direct Officer (正官) เสริม Resource → ตำแหน่งสูงในวงการวิชาการ/ราชการ
**Nemesis:** Wealth stars มากเกินไป — "财破印" ทำลายโอกาสการศึกษา

**เมื่อ Luck Pillar เข้า Officer element:** ชื่อเสียงและตำแหน่งสูงสุด — จุดพีค
**เมื่อ Luck Pillar เข้า Wealth:** ระวัง ถ้า Wealth มากจะทำลาย Resource

## 3. Follow Patterns (从格) — ยอมจำนนต่อสภาพแวดล้อม

> **หลักการ:** DM ไม่มีราก (rootless) อ่อนแอสุดขีด → ยอมจำนนต่อธาตุที่แข็งแกร่งที่สุดในดวง
> ห้าม "ช่วย" DM — การเสริม DM จะ **ทำลาย** Follow Pattern

### 3.1 从财格 — Follow Wealth

**เงื่อนไข:** Wealth stars ท่วมดวง DM ไม่มีราก ไม่มี Resource/Peers ช่วย
**เส้นทาง:** ชีวิตหมุนรอบทรัพย์และธุรกิจ — ความสำเร็จมาจากการ "ไหลตาม" โอกาสทางการเงิน ไม่ยึดติด
**Useful God:** Wealth + Output (ที่สร้าง Wealth) — ธาตุที่เสริม Wealth chain
**Nemesis:** Resource/Peers — "ช่วย" DM กลับทำให้ Pattern แตก = ชีวิตวุ่นวาย

### 3.2 从官杀格 — Follow Killings/Officer

**เงื่อนไข:** Officer/Killings ท่วมดวง DM ไม่มีราก ไม่มีสิ่งต้านทาน
**เส้นทาง:** ชีวิตถูก define ด้วยอำนาจและแรงกดดันภายนอก — ยิ่งยอมรับแรงกดดัน ยิ่งสำเร็จ
**Useful God:** Resource (ที่ transform Killings) + Wealth (เสริม Officer chain)
**Nemesis:** Output stars — ต้านแรงกดดัน ทำให้ Pattern แตก

### 3.3 从儿格 — Follow Output (Follow Son)

**เงื่อนไข:** Output stars (食神/傷官) ท่วมดวง DM ไม่มีราก ไม่มี Resource ต้าน
**เส้นทาง:** ชีวิตเดินตามพรสวรรค์และความคิดสร้างสรรค์ — ยิ่งให้ตัวเองแสดงออก ยิ่งสำเร็จ
**Useful God:** Output + Wealth (Output สร้าง Wealth = เส้นทางธรรมชาติ)
**Nemesis:** Resource — หยุดพลังงาน Output

### 3.4 从势格 — Follow Momentum

**เงื่อนไข:** DM อ่อนที่สุด ไม่มี element เดียวครอบงำ → หลายธาตุทำงานร่วมกัน
**เส้นทาง:** ชีวิตต้องการความยืดหยุ่นสูง — ปรับตัวเข้าหาธาตุที่แข็งแกร่งในช่วงนั้น ห้ามยึดติด
**Useful God:** ธาตุที่ "momentum" ของดวงชี้ไป — ขึ้นกับดวงเฉพาะ
**Nemesis:** การต้าน momentum = ความขัดแย้ง

## 4. Dominant Patterns (专旺格) — พลังธาตุเดียวบริสุทธิ์

> **หลักการ:** ธาตุเดียวครองดวงทั้งหมด → DM แข็งสุดขีด ไม่ต้องการ balance → ยิ่งเสริมธาตุนั้น ยิ่งดี

| Pattern | ชื่อจีน | เงื่อนไข | เส้นทางชีวิต | Nemesis |
|---------|---------|---------|-------------|---------|
| **曲直格** | Wood Dominant | 甲乙 DM + 寅卯辰 branches | ผู้นำด้านความเมตตา การศึกษา การเติบโต | Metal (ตัดไม้) |
| **炎上格** | Fire Dominant | 丙丁 DM + 寅午戌 branches | นักสื่อสาร ผู้นำ เจ้าเสน่ห์ | Water (ดับไฟ) |
| **稼穑格** | Earth Dominant | 戊己 DM + 辰戌丑未 branches | ผู้สะสม นักธุรกิจมั่นคง | Wood (ทำลายดิน) |
| **从革格** | Metal Dominant | 庚辛 DM + 申酉戌 branches | ผู้นำที่เด็ดขาด นักวิเคราะห์ | Fire (หลอมโลหะ) |
| **润下格** | Water Dominant | 壬癸 DM + 亥子丑 branches | นักปัญญาชน นักวิจัย นักเดินทาง | Earth (กั้นน้ำ) |

**เงื่อนไขพิเศษ:** Dominant Pattern แท้จริง (真专旺) ต้องไม่มีธาตุ Nemesis ในดวงเลย — ถ้ามีแม้เพียงหนึ่งเดียว = Flawed Dominant Pattern → ชีวิตขึ้นๆ ลงๆ

## 5. Pattern Purity และ Luck Pillar Activation — หลักการรวม

### กฎ Luck Pillar × Pattern

```
Pattern แข็งแกร่ง + LP เสริม Pattern = จุดพีคชีวิต
Pattern แข็งแกร่ง + LP ทำลาย Pattern = วิกฤต
Pattern อ่อน + LP เสริม = พัฒนาขึ้น
Pattern อ่อน + LP ทำลาย = ช่วงที่ยากมาก
```

### เมื่อ Useful God เข้าใน LP

- LP Stem + Branch เป็น Useful God element = **Activation Period** — โอกาสเปิด ชีวิตดีขึ้น
- Pattern ที่ Impure → ถ้า LP ขจัด Nemesis ออก = Pattern กลายเป็น Pure ชั่วคราว = ช่วงสำเร็จ
