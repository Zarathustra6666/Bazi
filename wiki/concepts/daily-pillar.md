# 流日 Daily Pillar — ดาวพลังประจำวัน

> พลังงานประจำวันที่กำหนดโดยวัฏจักรหกสิบ (六十甲子) — ชั้นเวลาที่เบาที่สุดในระบบ Bazi แต่ใช้ได้จริงเป็นตัวกรองคุณภาพของวันสำคัญ

## What it is

[[สี่เสา (Four Pillars)|Bazi]] ใช้วัฏจักรหกสิบคู่ (六十甲子 Sixty Jiazi Cycle) เป็นตัวนับวัน — ทุกวันในปฏิทินโลกมีคู่ [[กิ่งฟ้า (Heavenly Stems)|กิ่งฟ้า]] + [[ก้านดิน (Earthly Branches)|ก้านดิน]] กำกับอยู่ และวนซ้ำทุก 60 วัน

**โครงสร้าง:**
- Stem (กิ่งฟ้า) ของวัน → กำหนด [[สิบเทพ (Ten Gods)|Ten God]] เทียบกับ [[ธาตุเจ้าชะตา (Day Master)|Day Master]] (DM) ของเจ้าของดวง
- Branch (ก้านดิน) ของวัน → ตรวจสอบ 六冲 (Clash) และ 六合 (Harmony) กับเสาทั้ง 4 ในดวงกำเนิด
- ทั้งคู่ร่วมกันกำหนด "คุณภาพ" ของวันนั้นสำหรับบุคคลนั้นโดยเฉพาะ

**การคำนวณ:** ใช้สูตร Julian Day Number (JDN) โดยมีจุดอ้างอิงที่ JDN 2451545 = 1 มกราคม 2000 = 戊午 (วัฏจักรที่ 54)

## Why it matters

### น้ำหนักในลำดับชั้นเวลา

| ระดับ | ชื่อ | น้ำหนักสัมพัทธ์ | อุปมา |
|-------|------|----------------|-------|
| 大運 | Luck Pillar 10 ปี | สูงมาก | ภูมิอากาศ (Climate) |
| 流年 | Annual Luck | สูง | ฤดูกาล (Season) |
| 流月 | Monthly Luck | กลาง | พยากรณ์รายสัปดาห์ |
| **流日** | **Daily Luck** | **เบา** | **อุณหภูมิรายวัน** |

> [[luck-pillar-arc|LP 10-Year Arc]] ยืนยันว่า 大運 "holds infinitely more weight" กว่าทุกชั้น — 流日 จึงเป็นตัวกรองสุดท้าย ไม่ใช่ตัวกำหนดชะตาหลัก

### การใช้งานจริง

- **择日 (Date Selection):** ปรมาจารย์คลาสสิกใช้ 流日 เลือกวันมงคลสำหรับงานแต่งงาน ย้ายบ้าน เซ็นสัญญา
- **วันที่ควรหลีกเลี่ยง:** วันที่ Branch ของวัน 六冲 กับ Day Branch ของดวงกำเนิด = วันสั่นคลอนอัตลักษณ์/ความสัมพันธ์ → ไม่เหมาะตัดสินใจใหญ่
- **วันที่เอื้อ:** Stem/Branch ของวันตรงกับ [[เทพใช้งาน (用神)|Useful God]] = วันที่ปฏิบัติการตาม nature ดวง → ผลตอบแทนดีกว่าวันอื่น

## Key aspects

### การตีความ Stem ของวัน (Ten God)

Ten God ที่ Stem ของวันสร้างต่อ DM กำหนด "ธีม" ของวัน:

| Ten God ของ Day Stem | ธีมวัน |
|---------------------|--------|
| 比肩/劫财 (Peer/Rob Wealth) | วันสังสรรค์ แต่ระวังการแข่งขัน |
| 食神 (Eating God) | วันสร้างสรรค์ ผ่อนคลาย สุขสบาย |
| 傷官 (Hurting Officer) | วันแสดงออก/ความคิดเฉียบ แต่ขัดแย้งง่าย |
| 偏财/正财 (Wealth) | วันธุรกิจ เงิน การเจรจา |
| 七杀/正官 (Officer/7K) | วันงาน ความรับผิดชอบ แรงกดดัน |
| 偏印/正印 (Resource) | วันเรียนรู้ พักผ่อน รับการสนับสนุน |

### การตีความ Branch ของวัน (Clash / Harmony)

**六冲 กับเสาใดเสาหนึ่งในดวงกำเนิด:**

| เสาที่ถูก clash | Domain ที่กระทบ | ผลที่คาดวันนั้น |
|----------------|----------------|----------------|
| เสาวัน (Day Branch) | ตัวตน / คู่ครอง | ความไม่สบายใจส่วนตัว / เรื่องคู่ |
| เสาเดือน (Month Branch) | อาชีพ / พ่อแม่ | งานขัด / เจ้านายยุ่งยาก |
| เสาปี (Year Branch) | รากฐาน / ครอบครัว | ปัญหารอบข้าง |
| เสาชั่วโมง (Hour Branch) | แผนการ / ลูก | แผนเปลี่ยน / เหนื่อยกับอนาคต |

**六合 กับเสาใดเสาหนึ่ง:** วันที่พลังงานประสานกัน — เสริมความสัมพันธ์และความร่วมมือกับ domain นั้น

### น้ำหนักของ 六冲 เทียบตาม Domain

clash กับ Day Branch สำคัญที่สุดสำหรับความรู้สึกส่วนตัว เพราะ Day Pillar ใน Bazi คือ "ตัวตนและคู่ครอง" — วันที่ Branch ชนเสาวัน = วันที่รู้สึกไม่เป็นตัวเอง

## Open questions

- น้ำหนักสัมพัทธ์ของ 流月 vs 流日 ในกรณีที่ขัดแย้งกัน — ไม่มีมาตรฐานชัดเจนจากสำนักคลาสสิก
- 三合 ของ Branch วันกับเสาดวงกำเนิด — บางสำนักให้น้ำหนักพอกับ 六合 บางสำนักถือว่า daily 三合 ไม่นับ
- ขอบเขตของ "วัน" ในสำนัก Late-Zi (วันเปลี่ยนที่ 23:00 ไม่ใช่ 00:00)
