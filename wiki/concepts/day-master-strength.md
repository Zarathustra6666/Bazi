**Day Master Strength** คือการประเมินว่า [[ธาตุเจ้าชะตา (Day Master)]] มีพลังงานมากน้อยเพียงใดเมื่อเทียบกับสภาพแวดล้อมในดวง — เป็นตัวแปรกลางของ Modern Balance Method ในการหา [[เทพใช้งาน (用神)]] ถึงแม้ Classical method จะใช้ Month Commander เป็น primary แทน

## How It Works

### ปัจจัยที่ส่งผลต่อ DM Strength

**สนับสนุน DM (เพิ่มความแข็งแกร่ง):**
1. **Resource Gods (印星):** Direct Resource + Indirect Resource — generate DM
2. **Companion Gods (比劫):** Parallel + Rob Wealth — same element as DM
3. **Season (ฤดูกาล):** DM เกิดในฤดูกาลของตัวเอง (ธาตุ Wood แข็งสุดในฤดูใบไม้ผลิ)
4. **Rooting:** DM Stem มี Hidden Stems เดียวกันใน Branches

**บั่นทอน DM (ลดความแข็งแกร่ง):**
1. **Output Gods (食傷):** Food God + Hurting Officer — DM generate out (ถ่ายเทพลัง)
2. **Wealth Gods (財星):** Direct Wealth + Indirect Wealth — DM controls (ใช้พลัง)
3. **Officer Gods (官殺):** Direct Officer + 7-Killings — control DM (กดทับ)
4. **Season:** DM อยู่ใน Season ที่ตัวเองอ่อนแอ (Wood อ่อนสุดในฤดูใบไม้ร่วง)

---

### ประเภทของ DM Strength

**Strong DM (身強):** Resource + Companion มาก; เกิดใน Season ของตัวเอง; มีรากมั่นคง
→ **Modern Method:** ต้องการ Output, Wealth, Officer เพื่อ drain/control พลังส่วนเกิน

**Weak DM (身弱):** Output + Wealth + Officer มาก; เกิดนอก Season; ไม่มีราก
→ **Modern Method:** ต้องการ Resource, Companion เพื่อเสริมพลัง

**Very Strong / Very Weak — Special Cases:**
- **Extremely Strong (從旺格):** DM แข็งสุดขีด ไม่มีธาตุต่อต้าน → Follow Dominant Structure; Yong Shen = สิ่งที่เสริม DM ต่อไป (ขัด Balance logic)
- **Extremely Weak (從弱格/從格):** DM อ่อนมาก ไม่มีราก → Follow Structure; Yong Shen = ธาตุที่แข็งที่สุดในดวง (ยอมตาม)

---

### วิธีประเมิน: ระบบ Point Counting (Modern)

สำนักสมัยใหม่บางแห่งใช้ระบบให้คะแนน:
- Stem = น้ำหนักน้อย; Branch = น้ำหนักมากกว่า (เพราะมี Hidden Stems)
- Chief Hidden Stem > Middle Hidden Stem > Residual Hidden Stem
- Season Branch (Month Branch) = น้ำหนักสูงสุด

รวมคะแนน Supports vs Opponents → ถ้า Supports > Opponents: Strong; ถ้าน้อยกว่า: Weak

⚠️ ระบบนี้เป็น Modern approximation — ZPZQ ไม่ใช้ point system แต่ดู structure และ qualitative assessment

---

### ข้อวิจารณ์ของ Classical Method

[[子平真诠 Zi Ping Zhen Quan]] และสำนักคลาสสิกโต้แย้งว่า:
- DM Strength เพียงอย่างเดียวไม่พอในการกำหนด Yong Shen
- **"Strong DM ≠ ดี; Weak DM ≠ เลว"** — ขึ้นอยู่กับว่าดวงสามารถใช้ Gods ที่มีได้หรือไม่
- Month Commander กำหนดโครงสร้างก่อน; DM Strength เป็นแค่ปัจจัยหนึ่ง

ตัวอย่าง: Weak DM ที่มีราก + Month Commander ที่แข็ง → อาจ handle Wealth/Officer ได้ดี แม้ตาม Point System จะ "อ่อน"

## Why It Matters

DM Strength เป็น starting point ของ Modern Method ที่แพร่หลายที่สุดในโลก:
1. **ง่ายต่อการสอนและเรียน** — มีระบบ quantitative ชัดเจน
2. **ใช้กำหนด Yong Shen** ใน Modern approach
3. **กำหนดคุณภาพ Luck Pillars** — Luck ที่เสริม vs บั่นทอน DM
4. **เข้าใจพลวัตชีวิต** — Strong DM tend toward independence; Weak DM tend toward dependence on environment
