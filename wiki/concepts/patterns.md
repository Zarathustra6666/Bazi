# โครงสร้างดวงชะตา (格局)

> โครงสร้างหลักของดวงชะตาที่กำหนดศักยภาพและ [[เทพใช้งาน (用神)]] — ระบุจาก Ten God ใน Month Branch

## Overview
格局 (Gé Jú, Chart Structure / Patterns) คือระบบจำแนกโครงสร้างหลักของดวงชะตา [[Bazi (八字)]] โดยดูจาก [[สิบเทพ (十神)|Ten God]] ที่อยู่ใน Month Branch เป็นหลัก 格局 กำหนดว่าดวงนั้นมีศักยภาพแบบใด และ [[เทพใช้งาน (用神)]] ควรเป็น element ใด ระบบ格局 ได้รับการจัดระบบอย่างสมบูรณ์ที่สุดใน [[子平真诠 Zi Ping Zhen Quan]] โดย Shen Xiaozhan แห่งราชวงศ์ชิง

## Origin & History (ที่มา)
แนวคิด格局เริ่มปรากฏใน [[渊海子平 Yuan Hai Zi Ping]] สมัยซ่ง แต่ได้รับการพัฒนาและจัดระบบอย่างละเอียดที่สุดโดย **Shen Xiaozhan (沈孝瞻)** ในคัมภีร์ [[子平真诠 Zi Ping Zhen Quan]] สมัยชิง ซึ่งจำแนกดวงชะตาออกเป็น ~22 ประเภท ปัจจุบัน格局 System เป็นหัวใจของ Zi Ping School และเป็นสิ่งที่แยก Classical Bazi จาก Modern Western approach

## Core Definition (นิยามหลัก)

### วิธีกำหนด格局 (4 ขั้นตอน)
**ขั้นที่ 1:** ดู Ten God ที่อยู่ใน **Month Branch (月支)** เป็นจุดเริ่มต้น
**ขั้นที่ 2:** ตรวจว่า Ten God นั้น "transparent" (透出) ขึ้นมาใน Heavenly Stems หรือไม่
**ขั้นที่ 3:** ตัดสินว่าเป็น **Pure (清) หรือ Impure (浊)** — มีสิ่งรบกวนหรือไม่
**ขั้นที่ 4:** หา [[เทพใช้งาน (用神)]] ที่ "fulfill" Pattern นั้น

### ประเภท格局ทั้งหมด (~22 ประเภท)

**A. 8 Normal Patterns (八格) — ใช้กับดวงส่วนใหญ่**

| Pattern | ชื่อจีน | คำอธิบาย |
|---------|---------|---------|
| 建禄格 Jiàn Lù | Jian Lu | Day Master เจอ Jian Lu (ลักษณะพิเศษ) ในเดือน |
| 阳刃格 Yáng Rèn | Yang Blade | Yang Blade ใน Month Branch |
| 食神格 Shí Shén | Eating God | 食神 ใน Month Branch |
| 伤官格 Shāng Guān | Hurting Officer | 伤官 ใน Month Branch |
| 偏财格 Piān Cái | Indirect Wealth | 偏财 ใน Month Branch |
| 正财格 Zhèng Cái | Direct Wealth | 正财 ใน Month Branch |
| 七杀格 Qī Shā | Seven Killings | 七杀 ใน Month Branch |
| 正官格 Zhèng Guān | Direct Officer | 正官 ใน Month Branch |

**B. 4 Following Patterns (从格) — Day Master ยอมตาม dominant element**

| Pattern | เงื่อนไข |
|---------|---------|
| 从财格 Cóng Cái | DM อ่อนมาก Wealth element ครอบงำ |
| 从官格 Cóng Guān | DM อ่อนมาก Authority element ครอบงำ |
| 从儿格 Cóng Ér | DM อ่อนมาก Output element ครอบงำ |
| 从势格 Cóng Shì | DM อ่อนมาก สามธาตุแย่งชิงกัน |

**C. 5 Dominant Patterns (专旺格) — Day Master มีอำนาจสูงสุด**

| Pattern | ชื่อ | ธาตุ |
|---------|------|------|
| 曲直格 Qū Zhí | Wood Prosperity | ไม้ |
| 炎上格 Yán Shàng | Fire Ascending | ไฟ |
| 稼穑格 Jià Sè | Earth Consolidation | ดิน |
| 从革格 Cóng Gé | Metal Reform | โลหะ |
| 润下格 Rùn Xià | Water Descending | น้ำ |

**D. 5 Transformation Patterns (化格) — Stems transform**
เกิดจาก Stem Combination (天干合化) ที่สำเร็จเงื่อนไขครบถ้วน หายากมาก

## Five Elements Relationship (ความสัมพันธ์ 五行)
- Normal Patterns: แต่ละ Pattern มี element เฉพาะที่เป็น Useful God
- Follow Patterns: [[ธาตุเจ้าชะตา (Day Master)]] ยอม "ตาม" [[ธาตุทั้งห้า (Five Elements)]] ที่ครอบงำ — ใน Follow Pattern ต้องเสริม dominant element ไม่ใช่ DM
- Dominant Patterns: DM มีธาตุเดียวกันแทบทั้งดวง → ต้องการ element ที่เสริม ไม่ใช่ควบคุม

## Interaction Rules (กฎปฏิสัมพันธ์)

**Pure vs Impure Pattern:**
- **Pure (清):** Ten God ใน Month Branch ไม่ถูกรบกวน → ศักยภาพสูง
- **Impure (浊):** มีธาตุรบกวน Pattern → ศักยภาพลดลง
- Pattern ที่ Pure + Useful God แข็งแกร่ง = ดวงที่มีศักยภาพสูงสุด

**"Transparent" (透出):**
Ten God ใน Month Branch ที่ขึ้นมาใน Heavenly Stems ด้วย = Pattern แข็งแกร่งขึ้น

**สองแนวคิดหลักของ Zi Ping Zhen Quan:**
- **"Affection" (情):** พลวัตที่ถูกต้องระหว่าง Ten Gods ในดวง
- **"Strength" (力):** Elements มี Qi เพียงพอที่จะแสดงบทบาทของตน

## Interpretation Methods (วิธีตีความ)

Pattern ของดวงกำหนด:
1. **ศักยภาพด้านใด** ที่ดวงนั้นส่งเสริม (ความมั่งคั่ง อำนาจ ความฉลาด ฯลฯ)
2. **Useful God** คือ element ใด → ช่วง Luck Pillars ไหนดีที่สุด
3. **ความบริสุทธิ์** ของ Pattern บ่งบอกว่าศักยภาพนั้นจะเกิดขึ้นได้ง่ายแค่ไหน

## Schools of Thought (สำนักต่างๆ)

**Zi Ping School (ZPZQ-based):**
- กำหนด Pattern จาก Month Branch เสมอ
- Useful God = element ที่ fulfill Pattern
- Pattern บอกว่า DM จะ "ใช้" element ใด ไม่ใช่แค่ดู strong/weak

**Balance School:**
- ให้ความสำคัญกับ DM strength มากกว่า Pattern
- Useful God = element ที่สมดุล DM (อาจต่างจาก Zi Ping approach)

**ความขัดแย้ง:**
Follow Patterns และ Dominant Patterns เป็นประเด็นที่นักปฏิบัติถกเถียงมากที่สุด — เงื่อนไขในการตัดสินว่าดวงใด "Follow" แท้จริงไม่มีกฎตายตัว ต้องใช้ judgment สูง

## Common Misconceptions (ความเข้าใจผิด)

- ❌ "ดวงแข็ง (Strong) = ดวงดีเสมอ" — Follow Patterns ที่ DM อ่อนมากก็เป็นดวงดีได้
- ❌ "Follow Pattern เกิดขึ้นบ่อย" — ในความเป็นจริง Follow Patterns พบน้อยมาก เงื่อนไขต้องครบถ้วนมาก
- ❌ "Pattern กำหนดชะตาตายตัว" — Pattern บอกศักยภาพ แต่ Luck Pillars กำหนดว่าจะเกิดขึ้นเมื่อไร
- ❌ "ทุกดวงมี Pattern ที่ชัดเจน" — บางดวง Pattern ไม่ชัด (Mixed/Impure) ต้องใช้ judgment สูง
