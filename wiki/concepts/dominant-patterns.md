# Dominant Patterns (专旺格)

> Pattern พิเศษที่ตรงข้ามกับ [[Follow Patterns (从格)]] — [[ธาตุเจ้าชะตา (Day Master)]] ไม่ได้ยอมตาม แต่ **ครองอำนาจสูงสุด** ธาตุ DM เดียวล้นดวงทั้งดวง ไม่มีธาตุใดกล้าต้าน

---

## หลักการ

ใน Dominant Pattern ธาตุ DM เดียวครอบงำดวงอย่างสมบูรณ์ — Useful God คือธาตุที่ **เสริม DM ต่อไป** ไม่ใช่ธาตุที่ควบคุม DM สิ่งที่ "ควบคุม" DM ปกติกลับกลายเป็น nemesis ที่ทำลายดวง

**เงื่อนไขร่วม:**
1. [[ธาตุเจ้าชะตา (Day Master)]] ต้องมี Stem ตรงกับธาตุ dominant
2. [[ก้านดิน (Earthly Branches)|Earthly Branches]] ต้องก่อ directional set หรือ three-harmony combination ของธาตุนั้น
3. ธาตุที่ "ควบคุม" (controlling element) ต้องไม่ปรากฏในทั้ง Stems และ Branches

---

## ทั้ง 5 ประเภท

### 1. 曲直格 (Qū Zhí Gé) — Wood Dominant

| เงื่อนไข | รายละเอียด |
|---------|-----------|
| DM | 甲 (Jia) หรือ 乙 (Yi) — Wood |
| Branch combo | **Directional:** 寅(ขาล) + 卯(เถาะ) + 辰(มะโรง) — East Wood trio |
| | **Three-harmony:** 亥(กุน) + 卯(เถาะ) + 未(มะแม) — Wood three-harmony |
| ห้ามมี | Metal — ธาตุโลหะในทั้ง Stems และ Branches |
| Useful Gods | Water (สร้าง Wood), Wood (เสริม), Fire (ผลผลิต) |

**ผลในชีวิต:** ความเจริญงอกงาม ความเมตตา ความเป็นผู้นำที่อ่อนโยน การเติบโตอย่างต่อเนื่อง

---

### 2. 炎上格 (Yán Shàng Gé) — Fire Dominant

| เงื่อนไข | รายละเอียด |
|---------|-----------|
| DM | 丙 (Bing) หรือ 丁 (Ding) — Fire |
| Branch combo | **Directional:** 巳(มะเส็ง) + 午(มะเมีย) + 未(มะแม) — South Fire trio |
| | **Three-harmony:** 寅(ขาล) + 午(มะเมีย) + 戌(จอ) — Fire three-harmony |
| ห้ามมี | Water — ธาตุน้ำในทั้ง Stems และ Branches |
| Useful Gods | Wood (เชื้อเพลิง), Fire (เสริม), Earth (ผลผลิต) |

**ผลในชีวิต:** ความฉายแสง การแสดงออก ความกล้าหาญ ความสร้างสรรค์ระดับสูง

---

### 3. 稼穡格 (Jià Sè Gé) — Earth Dominant

| เงื่อนไข | รายละเอียด |
|---------|-----------|
| DM | 戊 (Wu) หรือ 己 (Ji) — Earth |
| Branch combo | ≥ 3 ใน 4 Earth branches: 辰(มะโรง) + 戌(จอ) + 丑(ฉลู) + 未(มะแม) |
| | Fire support เพิ่มความแข็งแกร่ง |
| ห้ามมี | Wood — ธาตุไม้ในทั้ง Stems และ Branches |
| Useful Gods | Fire (สร้าง Earth), Earth (เสริม), Metal (ผลผลิต) |

**ผลในชีวิต:** ความมั่นคงสูงสุด การสะสมทรัพย์ ความน่าเชื่อถือ ผู้ค้ำจุนผู้อื่น

---

### 4. 从革格 (Cóng Gé Gé) — Metal Dominant

| เงื่อนไข | รายละเอียด |
|---------|-----------|
| DM | 庚 (Geng) หรือ 辛 (Xin) — Metal |
| Branch combo | **Directional:** 申(วอก) + 酉(ระกา) + 戌(จอ) — West Metal trio |
| | **Three-harmony:** 巳(มะเส็ง) + 酉(ระกา) + 丑(ฉลู) — Metal three-harmony |
| ห้ามมี | Fire — ธาตุไฟในทั้ง Stems และ Branches |
| Useful Gods | Earth (สร้าง Metal), Metal (เสริม), Water (ผลผลิต) |

**ผลในชีวิต:** ความเด็ดขาด การปฏิรูป ความยุติธรรม ความสามารถด้านกฎหมาย/วิศวกรรม

---

### 5. 润下格 (Rùn Xià Gé) — Water Dominant

| เงื่อนไข | รายละเอียด |
|---------|-----------|
| DM | 壬 (Ren) หรือ 癸 (Gui) — Water |
| Branch combo | **Directional:** 亥(กุน) + 子(ชวด) + 丑(ฉลู) — North Water trio |
| | **Three-harmony:** 申(วอก) + 子(ชวด) + 辰(มะโรง) — Water three-harmony |
| ห้ามมี | Earth — ธาตุดินในทั้ง Stems และ Branches |
| Useful Gods | Metal (สร้าง Water), Water (เสริม), Wood (ผลผลิต) |

**ผลในชีวิต:** ปัญญา ความยืดหยุ่น การหล่อเลี้ยง ความสามารถด้านการวิจัย/การเดินทาง
