# Pillar Domain Rules — ความหมายเฉพาะของแต่ละเสา

> คัมภีร์คลาสสิก: *"เสาปีคือราก เสาเดือนคือต้นอ่อน เสาวันคือดอกไม้ เสาเวลาคือผลไม้"* — แต่ละเสามีอาณาเขตชีวิตของตัวเอง

---

## 1. หลักการอ่านแต่ละเสา

[[สี่เสา (Four Pillars)|สี่เสา]] ไม่ได้แค่ให้ข้อมูล — แต่ละเสา **ปกครองช่วงชีวิตและความสัมพันธ์ที่ต่างกัน**:

| เสา | ภาษาจีน | ช่วงอายุ (แนวทาง) | อาณาเขตชีวิต | ภาพเปรียบ |
|-----|--------|---------|------------|---------|
| **เสาปี** (Year) | 年柱 | 0–16 ปี | บรรพบุรุษ, สภาพแวดล้อมเกิด, รากฐานสังคม | ราก (根) |
| **เสาเดือน** (Month) | 月柱 | 17–35 ปี | พ่อแม่, อาชีพ, ชื่อเสียง, Pattern ดวงชะตา | ต้นอ่อน (苗) |
| **เสาวัน** (Day) | 日柱 | 36–51 ปี | ตัวตนแท้จริง, คู่ครอง, ชีวิตกลางอายุ | ดอกไม้ (花) |
| **เสาเวลา** (Hour) | 时柱 | 52+ ปี | ลูก, มรดก, ความสำเร็จบั้นปลาย | ผล (果) |

> **หมายเหตุ:** ช่วงอายุเป็น "แนวทาง" ไม่ใช่กฎตายตัว — Luck Pillars เป็นตัว กำหนดช่วงเวลาที่แม่นยำกว่า

---

## 2. เสาปี (年柱) — รากฐานและบรรพบุรุษ

### สิ่งที่เสาปีบอก

- **บรรพบุรุษ (祖先):** ฐานะและชื่อเสียงของครอบครัว พ่อแม่ ปู่ย่าตายาย
- **สภาพแวดล้อมเกิด:** ความมั่งคั่ง/ความยากจนของครอบครัว บ้านเกิด สังคมที่เติบโต
- **ความสัมพันธ์กับผู้อาวุโส:** เจ้านาย บุคคลที่มีอำนาจ
- **ช่วง 0–16 ปี:** ชีวิตวัยเด็ก การศึกษาขั้นต้น ครอบครัวต้นกำเนิด

### อ่าน Year Stem + Branch

**Year Stem** (天干เสาปี):
- [[สิบเทพ (十神)|Ten God]] ของ Year Stem เทียบ [[ธาตุเจ้าชะตา (Day Master)|Day Master]]:
  - Direct Officer/Seven Killings = บรรพบุรุษมีอำนาจ/มีชื่อเสียง
  - Resource = ครอบครัวมีการศึกษาดี, ได้รับการสนับสนุน
  - Wealth = บรรพบุรุษมีทรัพย์
  - Peers/Rob Wealth = พี่น้องมาก, การแข่งขันในครอบครัว

**Year Branch** (地支เสาปี):
- Hidden Stems บอกรายละเอียดเพิ่มเติมเกี่ยวกับ "ดิน" ที่หยั่งราก
- Year Branch ชนกับ Day Branch (日年相冲) = ความตึงเครียดระหว่างตัวตนและรากฐาน

### ตัวอย่างการอ่าน

| Year Pillar TG | ความหมาย |
|---------------|---------|
| 正官 (Direct Officer) | ปู่ย่า/พ่อแม่มีตำแหน่งสังคม, เติบโตในสภาพแวดล้อมที่มีกฎระเบียบ |
| 偏财 (Indirect Wealth) | บรรพบุรุษมีธุรกิจ, เติบโตท่ามกลางการค้าขาย |
| 七杀 (Seven Killings) | ครอบครัวเข้มแข็งแต่มีแรงกดดัน, อาจมีความขัดแย้งในวัยเด็ก |
| 正印 (Direct Resource) | บรรพบุรุษมีการศึกษา, วัยเด็กได้รับการสนับสนุนด้านความรู้ |
| 比肩/劫财 (Peers) | พี่น้องมาก, ต้องแข่งขันเพื่อความสนใจของพ่อแม่ |

## 3. เสาเดือน (月柱) — อาชีพและ Pattern

### สิ่งที่เสาเดือนบอก

เสาเดือนเป็น **เสาที่สำคัญที่สุด** ในดวงชะตา เพราะ:
1. **กำหนด Pattern (格局):** Main Qi ของ Month Branch = Pattern ของดวง (ดู [[格局 Pattern Life Narratives]])
2. **บอกเส้นทางอาชีพ:** Ten God ของ Month Stem = รูปแบบการทำงาน
3. **พ่อแม่:** Month Branch = Parents Palace (宫)
4. **ช่วง 17–35 ปี:** ช่วงการสร้างตัว, การศึกษาระดับสูง, อาชีพแรก

### อ่าน Month Stem + Branch

**Month Stem** (天干เสาเดือน):
| TG ของ Month Stem | ความหมายอาชีพ |
|-----------------|------------|
| 正官 Direct Officer | งานองค์กร/ราชการ — เส้นทางตามระบบ |
| 七杀 Seven Killings | งานที่มีความกดดันสูง — ผู้นำ/การแข่งขัน |
| 正财 Direct Wealth | งานที่มีรายได้มั่นคง — การเงิน/ธุรกิจประจำ |
| 偏财 Indirect Wealth | งานธุรกิจ/การค้า — รายได้หลายทาง |
| 食神 Eating God | งานสร้างสรรค์/บริการ — ทำในสิ่งที่รัก |
| 傷官 Hurting Officer | งานที่ใช้ความคิดและการแสดงออก — ศิลปะ/วิชาการ |
| 正印 Direct Resource | งานวิชาการ/การศึกษา — ความก้าวหน้าผ่านความรู้ |
| 偏印 Indirect Resource | งานเฉพาะทาง/นอกกระแส — ผู้เชี่ยวชาญพิเศษ |
| 比肩 Companion | ทำงานอิสระ — ต้องสร้างตัวเอง |
| 劫财 Rob Wealth | งานที่มีความเสี่ยงสูง — ระวังหุ้นส่วน |

**Month Branch (地支เสาเดือน):**
- Hidden Stems = ทักษะ "ซ่อนเร้น" ที่ใช้ในอาชีพ
- Month Branch ชนกับ Day Branch = ความขัดแย้งระหว่างอาชีพและชีวิตส่วนตัว

### ความสัมพันธ์กับพ่อแม่

- Month Branch = Parents Palace: ดู Ten God ของ Month Stem ว่าคืออะไรต่อ DM
  - Resource = แม่สนับสนุนดี, ได้รับการเลี้ยงดูดี
  - Wealth = พ่อมีฐานะดี
  - Officer = พ่อมีอำนาจ, ครอบครัวมีระเบียบวินัย
  - Peers = พ่อแม่มีพลังงานคล้าย DM แต่อาจแข่งขันกัน

## 4. เสาวัน (日柱) — ตัวตนและคู่ครอง

### สิ่งที่เสาวันบอก

เสาวันคือ **แกนกลางของดวงชะตา**:
- **Day Stem** = [[ธาตุเจ้าชะตา (Day Master)|Day Master (ธาตุเจ้าชะตา)]] — ตัวตนแท้จริง
- **Day Branch** = Spouse Palace (夫妻宫) — คุณสมบัติคู่ครอง บรรยากาศการแต่งงาน
- **ช่วง 36–51 ปี:** ชีวิตกลางอายุ, การแต่งงาน, ความสำเร็จส่วนตัว

### อ่าน Day Branch (Spouse Palace)

**ดวงชาย — Day Branch บอกคุณสมบัติภรรยา:**
| Day Branch | ภาพภรรยา |
|-----------|---------|
| 子 Water Rat | ฉลาด ปรับตัวได้ สื่อสารเก่ง |
| 午 Fire Horse | กระตือรือร้น มีพลังงาน อาจดื้อ |
| 卯 Wood Rabbit | ละเอียดอ่อน อ่อนโยน รักศิลปะ |
| 酉 Metal Rooster | มีระเบียบ สวยงาม มาตรฐานสูง |

**ดวงหญิง — Day Branch บอกคุณสมบัติสามี:**
- ดู Ten God ของ Day Branch Hidden Stem ที่เป็น Officer/Killings สำหรับดวงหญิง

### สัญญาณในเสาวัน

| สัญญาณ | ความหมาย |
|-------|---------|
| Day Branch = 空亡 (Void) | คู่ครองอาจไม่อยู่ด้วยนาน หรือความห่างเหิน |
| Day Branch ชนกับ Month Branch | งานกับชีวิตส่วนตัวขัดแย้ง |
| Day Branch ชนกับ Year Branch | ความตึงเครียดระหว่างครอบครัวเก่าและใหม่ |
| Hidden Stem ใน Day Branch ไม่มี Officer/Killings (ดวงหญิง) | การแต่งงานช้า หรือไม่เน้นการแต่งงาน |
| Hidden Stem ใน Day Branch ไม่มี Wealth (ดวงชาย) | อาจต้องรอนาน หรือคู่ครองไม่โดดเด่นด้าน Wealth |

## 5. เสาเวลา (时柱) — ลูกหลานและมรดก

### สิ่งที่เสาเวลาบอก

- **ลูกหลาน (子女):** คุณสมบัติลูก ความสัมพันธ์กับลูก จำนวนลูก
- **ความสำเร็จบั้นปลาย:** ความสำเร็จของตัวเองในช่วงอายุ 52+
- **มรดก:** สิ่งที่ทิ้งไว้ให้ generation ถัดไป
- **ช่วง 52+ ปี:** ผลของสิ่งที่สร้างมาตลอดชีวิต

### อ่าน Hour Stem (ดูลูก)

**ดวงชาย — Output stars = ลูก:**
- 食神 (Eating God) = ลูกที่น่ารัก มีความสุข มีพรสวรรค์
- 傷官 (Hurting Officer) = ลูกฉลาดมาก แต่อาจเถียงเก่ง

**ดวงหญิง — Officer/Killings = ลูก (traditional view):**
- 正官 (Direct Officer) = ลูกมีระเบียบ มีเกียรติ
- 七杀 (Seven Killings) = ลูกแข็งแกร่ง ท้าทาย

### อ่าน Hour Branch

| Hour Branch TG | ความหมายบั้นปลาย |
|--------------|--------------|
| 正官/七杀 | มีชื่อเสียงและอำนาจในบั้นปลาย |
| 正财/偏财 | ทรัพย์สะสมดีในบั้นปลาย |
| 食神/正印 | ความสุขสงบ ได้รับการดูแลจากลูก |
| 傷官/偏印 | ความสำเร็จจากความรู้พิเศษ |
| 比肩/劫财 | ต้องพึ่งตัวเอง ลูกเป็นอิสระ |

## 6. กฎปฏิสัมพันธ์ระหว่างเสา

### ลำดับความแข็งแกร่ง (ตาม ZPZQ)

```
เสาเดือน > เสาวัน > เสาปี > เสาเวลา
```
เสาเดือนมีน้ำหนักสูงสุดในการกำหนด Pattern และ Useful God

### การชน (冲) ระหว่างเสา — ผลกระทบต่อชีวิต

| การชน | ผลกระทบ |
|------|--------|
| Year ชน Month | ความขัดแย้งระหว่างรากฐานและอาชีพ |
| Month ชน Day | งานกระทบชีวิตส่วนตัว/การแต่งงาน |
| Day ชน Hour | ความขัดแย้งกับลูก หรือความไม่ลงรอยในบั้นปลาย |
| Year ชน Day | ความขัดแย้งระหว่างตัวตนและรากฐานครอบครัว |

### การมองหา "Ten God ที่ซ้ำกันหลายเสา"

ถ้า Ten God เดียวกันปรากฏในหลายเสา → **Theme นั้นเด่นชัดในชีวิต**:
- Officer ใน Month + Year = ชีวิตเชื่อมกับอำนาจและระบบตลอด
- Resource ใน Month + Hour = การศึกษาและความรู้สำคัญทั้งต้นและปลายชีวิต
- Wealth ใน Day + Hour = ทรัพย์สะสมมาจากกลางอายุและส่งต่อในบั้นปลาย

## 7. วิธีสังเคราะห์ทั้งสี่เสา

**ขั้นตอนอ่าน:**

1. **Year Pillar:** "รากมาจากไหน?" — ดู TG ของ Year Stem + Branch Hidden Stems
2. **Month Pillar:** "Pattern คืออะไร? เส้นทางอาชีพ?" — หา Pattern + TG ของ Month Stem
3. **Day Pillar:** "ตัวตนแท้จริงและคู่ครอง?" — DM + Day Branch Spouse Palace
4. **Hour Pillar:** "ลูกและมรดก?" — TG ของ Hour Stem/Branch

**ประโยคสังเคราะห์ตัวอย่าง:**
> "คุณเกิดในครอบครัวที่ [Year TG meaning] เส้นทางอาชีพของคุณคือ [Month TG + Pattern] คู่ครองจะเป็น [Day Branch meaning] และบั้นปลายคุณจะ [Hour TG meaning]"
