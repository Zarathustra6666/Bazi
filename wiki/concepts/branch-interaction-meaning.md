# Branch Interactions × Pillar Pairs — ความหมายชีวิตของการชนและรวมกันระหว่างเสา

> หลักปรมาจารย์: **ประเภทของ interaction + คู่เสาที่เกิด = ความหมายชีวิตที่ระบุได้** — 六冲 ระหว่าง Day-Hour ไม่ใช่เรื่องเดียวกับ 六冲 ระหว่าง Year-Month อย่าอ่านรวมกัน

---

## 1. Life Domain Map — ความหมายแต่ละเสา

| เสา | Pillar | Palace | ชีวิตที่เกี่ยวข้อง |
|-----|--------|--------|-----------------|
| 年柱 Year | YP | บรรพบุรุษ/สังคม | รากฐาน ตระกูล ชื่อเสียงสังคม วัยเด็ก |
| 月柱 Month | MP | อาชีพ/เจ้านาย | การงาน ผู้บังคับบัญชา สภาพแวดล้อมทำงาน |
| 日柱 Day | DP | ตัวตน/คู่ครอง (配偶宫) | ตัวเอง บุคลิก คู่ครอง ชีวิตแต่งงาน |
| 时柱 Hour | HP | ลูก/มรดก (子女宫) | ลูกหลาน มรดก ตัวตนภายใน ชีวิตบั้นปลาย |

**หลักการ:** Day Branch = Spouse Palace โดยตรง, Hour Branch = Children Palace โดยตรง

## 2. ประเภท Branch Interaction

### 2.1 六冲 (Six Clashes) — การชน: ความขัดแย้งและการเปลี่ยนแปลง

คู่ที่ชน: 子午, 丑未, 寅申, 卯酉, 辰戌, 巳亥

**ธรรมชาติ:** ความไม่มั่นคง การเปลี่ยนแปลง บางครั้งดี (หยุดนิ่งถูกทำลาย) บางครั้งร้าย (สิ่งดีถูกทำลาย)

### 2.2 六合 (Six Harmonies) — ซ่อนเร้น: ความร่วมมือลึก

คู่ที่รวม: 子丑合土, 寅亥合木, 卯戌合火, 辰酉合金, 巳申合水, 午未合火

**ธรรมชาติ (暗合):** "ซ่อนเร้น" — ความสัมพันธ์ที่ไม่ชัดเจน แต่มีพลังแฝง ความช่วยเหลือที่ไม่ได้ขอ

### 2.3 三合 (Three Harmonies) — เปิดเผย: สนับสนุนและขยาย

สามกลุ่ม: 申子辰 (Water), 亥卯未 (Wood), 寅午戌 (Fire), 巳酉丑 (Metal)

**ธรรมชาติ (明合):** "เปิดเผย" — ความร่วมมือที่เห็นได้ชัด การขยายธาตุ การสนับสนุนจากภายนอก

## 3. Pillar Pair × Clash/Harmony Matrix (ความหมายชีวิต)

### 3.1 Day Pillar — Hour Pillar (日时)

| Interaction | ความหมายชีวิต | ระดับผลกระทบ |
|-------------|--------------|-------------|
| **日时 六冲** | ⚠️ ตัวตนกับ legacy/ลูก ขัดแย้งกัน — ปัญหาแต่งงานรุนแรง มีความขัดแย้งตลอดชีวิต + อาจกระทบลูก (สุขภาพ/ความสัมพันธ์กับลูก) | สูงมาก |
| **日时 六合** | ตัวตนกับลูก/legacy "เชื่อมกันเงียบๆ" — มีความสุขลึกๆ กับชีวิตครอบครัว ลูกเป็นที่พึ่งยามแก่ | ดี |
| **日时 三合** | ตัวตนและ legacy ไปทิศเดียวกัน — เสริมพลังซึ่งกันและกัน โชคดีเรื่องลูก | ดีมาก |

**คำอธิบายปรมาจารย์ (日时 六冲):**
> Day Branch = Spouse Palace + Hour Branch = Children Palace → ถ้าทั้งสองชนกัน = "ดอกไม้และผลไม้ขัดแย้งกัน" → ความไม่สงบในครอบครัวตลอด ปัญหาแต่งงานหนัก บางครั้งส่งผลถึงลูก

### 3.2 Month Pillar — Day Pillar (月日)

| Interaction | ความหมายชีวิต | ระดับผลกระทบ |
|-------------|--------------|-------------|
| **月日 六冲** | อาชีพ/สภาพแวดล้อมทำงาน ขัดแย้งกับ "ตัวตน" — ความตึงเครียดกับเจ้านาย/องค์กร ตัวเองไม่ fit กับ culture งาน | สูง |
| **月日 六合** | อาชีพและตัวตน "ซ่อนเร้นเชื่อมกัน" — งานที่ทำ reflect ตัวตนลึกๆ ความสุขซ่อนในอาชีพ | ดี |
| **月日 三合** | อาชีพและตัวตนไปในทิศเดียวกัน — ทำงานที่ตัวเองรัก เจ้านายสนับสนุน ชีวิตงานราบรื่น | ดีมาก |

## 4. Insights จาก NotebookLM (40-source Comparative Notebook)

**ZPZQ — "การรวม Resolve การชนได้":**
ตำรา Zi Ping Zhen Quan ย้ำว่า "การชนและการลงโทษไม่ใช่เรื่องสวยงาม แต่การรวม (Harmony) สามารถสลายสิ่งเหล่านี้ได้" — อย่างไรก็ตาม การรวมเพื่อแก้การชนในที่หนึ่ง อาจสร้างปัญหาใหม่ในอีกที่หนึ่ง ต้องดูโครงสร้างรวมเสมอ

**Day-Hour Clash (วัน-เวลา) — ผลที่ระบุชัด:**
- ผลต่อคู่ครอง: อาจนำไปสู่ **"แยกกันอยู่หลังแต่งงาน"** หรือความตึงเครียดต่อเนื่อง
- ผลต่อบุตร: สะท้อนความขัดแย้งระหว่าง "ความต้องการตนเอง vs หน้าที่ต่อบุตร"
- ตัวอย่าง: **卯酉冲** มักเกี่ยวกับความขัดแย้งในคุณค่าที่กระทบความสัมพันธ์ใกล้ชิด

**Month-Day Clash (เดือน-วัน) — ผลที่ระบุชัด:**
- มักแสดงออกเป็น **"เปลี่ยนงานบ่อย"** หรือเปลี่ยนทิศทางชีวิตอย่างฉับพลัน
- **寅申冲**: ความขัดแย้งระหว่างการขยายตัว (Wood instinct) กับกฎระเบียบ (Metal control) ในที่ทำงาน

**Year-Month Harmony (三合) — ผลต่อรากฐาน:**
- สร้างทิศทางธาตุทรงพลัง — **"ชีวิตถูกขับเคลื่อนด้วยธีมธาตุนั้นตั้งแต่เด็ก"**
- ตัวอย่าง: 申子辰 ใน Year+Month+LP → เสริมพลังน้ำอย่างมาก → สติปัญญา/อารมณ์เป็นธีมหลัก

**Year-Month Harmony (六合) — พันธนาการ:**
- อาจหมายถึงตัวตนทางสังคม (Month) "ผูกติดกับรากเหง้า/ความคาดหวังของตระกูล (Year) อย่างแยกไม่ออก" — เกื้อกูลหรือ **สูญเสียความเป็นอิสระ**

**จุดบอดของ modern school:** นักวิจารณ์ชี้ว่าสำนักสมัยใหม่มักมองข้ามรายละเอียด Palace-Clash แล้วไปเน้น DM Strength เพียงอย่างเดียว → พลาดรายละเอียดสำคัญของเหตุการณ์ในแต่ละช่วงชีวิต

## 5. ตัวอย่างจากการอ่านดวงจริง (Narrative Prose notebook — Sean Chan)

**Chen-Xu Clash (辰戌冲) — "ลบ Void":**
ในกรณีศึกษาจริง: 辰戌冲 ปี 2018 ถูกตีความว่า "removing the Void (空亡)" ใน Year Pillar → **activate ส่วนของดวงที่ dormant อยู่** → เปิดโอกาสใหม่ที่เคยถูกปิดกั้น
- **บทเรียน:** ไม่ใช่ทุก Clash = ร้าย — บางครั้ง Clash เปิดประตูที่ปิดอยู่

**Si-Shen Penalty (巳申刑) — ผลต่อความสัมพันธ์:**
巳申刑 ทำลาย "Metal buffer" ใน Month Branch → **"relationship issues surfacing"** ทันที
- **บทเรียน:** Punishment (刑) บางครั้งรุนแรงกว่า Clash เพราะทำลาย element ที่ป้องกันอยู่

**Chen-Chen Self-Punishment (辰辰自刑) — ผลต่อ Spouse Palace:**
辰辰自刑 (เมื่อ Annual Luck นำ 辰 เข้ามาชนกับ 辰 ใน Day Branch) → "harmed Day Branch and Spouse Element directly" → ความขัดแย้งหรือความไม่มั่นคงในชีวิตคู่
- **บทเรียน:** Self-Punishment (自刑) จาก LP/Annual กระทบ Spouse Palace โดยตรง

**"Wealth ใน Branch = เติบโต" (Anita Rosenberg):**
เมื่อ Wealth star ปรากฏใน **Branches** (ก้านดิน) ไม่ใช่แค่ Stems → หมายถึงทรัพย์ที่ **"grows"** (เติบโตเรื่อยๆ) ต่างจาก Wealth ใน Stem ที่เห็นชัดแต่ไม่มีราก

## 6. LP/Annual Branch กับ Chart Pillars

เมื่อ **LP Branch** หรือ **Annual Luck Branch** ชนหรือรวมกับ Chart Pillar branches:

| Trigger (LP/Annual Branch) | Chart Pillar ที่ Clash | ผล |
|---------------------------|----------------------|-----|
| Clash Day Branch | 配偶宫 ถูกรบกวน | ปัญหาแต่งงาน/คู่ครองในช่วงนั้น |
| Clash Month Branch | สภาพแวดล้อมการงานสั่นสะเทือน | การเปลี่ยนงาน การถูกกดดัน |
| Clash Hour Branch | 子女宫 ถูกรบกวน | ปัญหากับลูก หรือ มรดกที่สร้างถูกท้าทาย |
| Clash Year Branch | ชื่อเสียง/รากฐานถูกกระทบ | ชื่อเสียงสังคมเปลี่ยน |
| Harmony Day Branch | 配偶宫 ได้รับพลัง | โอกาสพบคู่ครอง/ความสัมพันธ์ดีขึ้น |
| Three-Harmony forms with Chart | ธาตุนั้นแข็งแกร่งมาก | ถ้าธาตุนั้น = Useful God → ยุคทอง |

## 6. ความแตกต่าง 三合 vs 六合 — ปรมาจารย์อ่านอย่างไร

| ลักษณะ | 三合 (Three Harmony) | 六合 (Six Harmony) |
|--------|---------------------|-------------------|
| ประเภท | 明合 (Open/Visible) | 暗合 (Hidden/Latent) |
| แหล่งสนับสนุน | ภายนอก — สังคม อาชีพ เครือข่าย | ภายใน — อารมณ์ ความสัมพันธ์ลึก |
| ความเร็วผล | เร็ว เห็นได้ชัด | ช้า แต่ยั่งยืน |
| ต้องการ Branch ครบ | ต้องการ 3 Branch (2 ใน Chart + 1 ใน LP) | 2 Branch พอ |
| ผลเมื่อครบ | ธาตุใหม่ถูกสร้าง (transformation) | ความสัมพันธ์ถูกเสริม |
