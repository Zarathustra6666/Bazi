# David Yek (泉緣 Quan Yuan)

> นักปฏิบัติ Bazi อิสระชาวมาเลเซียในนาม Quan Yuan (泉緣) — ผู้สืบทอดสายวิชา YangGong GuFa และผู้พัฒนา "Bird Profiles" สำหรับ [[สิบเทพ (十神)]]

## Overview

David Yek ดำเนินงานผ่าน **Quan Yuan (泉緣)** ซึ่งแปลว่า "ต้นธาร" — สะท้อนแนวคิดว่าสืบทอดจากต้นกำเนิดดั้งเดิม เขาเน้นตัวเองว่าเป็น "self-sufficient practitioner" ที่มองวิชานี้เป็นการกุศลทางการศึกษา ไม่ใช่อาชีพหาเงิน เขาอ้างสายวิชาจากปรมาจารย์ผู้ล่วงลับ **Li DingXing** และระบบ **YangGong GuFa (ยางกงวิธีโบราณ)** ซึ่งเป็นระบบ Feng Shui/Bazi แบบดั้งเดิม David Yek เป็นที่รู้จักในโลกอินเทอร์เน็ตภาษาอังกฤษผ่าน blog "David2u" ที่มีบทความเชิงวิเคราะห์เชิงลึกจำนวนมาก

## Key Facts

- **แบรนด์:** Quan Yuan (泉緣) / David2u
- **สายวิชา:** Li DingXing → YangGong GuFa; ยึดคัมภีร์ [[子平真诠 Zi Ping Zhen Quan|ZPZQ]], [[滴天髓 Di Tian Sui|DTS]], [[穷通宝鑑 Qiong Tong Bao Jian|QTBJ]] เป็นหลัก
- **รายได้:** Workshop และ Consultancy — ไม่ใช่ high-fee academy
- **จุดยืน:** Feng Shui ในฐานะ "Qi Flow Paradigm" — ถ้าวัด Qi ไม่ได้ การจัดการก็เป็น "voodoo"
- **ภูมิประเทศ:** เชื่อมโยง Bazi เข้ากับสถาปัตยกรรมและ Qi Flow ในพื้นที่จริง

## Bird Profiles System (ระบบนก)

David Yek พัฒนาการเรียกชื่อ [[สิบเทพ (十神)]] ด้วย "นก" เพื่อหลีกเลี่ยงความสับสนข้ามสำนัก:

| นก | สิบเทพ |
|----|--------|
| Phoenix (ฟีนิกซ์) | Direct Wealth (正財) |
| Dodo | Indirect Wealth (偏財) |
| Dove (นกพิราบ) | Direct Officer (正官) |
| Vulture (อีแร้ง) | 7 Killings (七殺) |
| Sparrow (กระจอก) | Rob Wealth (劫財) |
| Eagle (นกอินทรี) | Friend/Companion (比肩) |
| Peacock (นกยูง) | Eating God (食神) |
| Parrot (นกแก้ว) | Hurting Officer (傷官) |
| Ostrich (นกกระจอกเทศ) | Direct Resource (正印) |
| Owl (นกฮูก) | Indirect Resource (偏印) |

*หมายเหตุ: ต่างจาก DOPE system ของ [[Joey Yap (杨登亮)]] ซึ่งใช้นกเดียวกัน (Dove/Owl/Peacock/Eagle) แต่หมายถึง Big 4 personality types — ไม่ใช่ Ten Gods ครบ 10*

## Key Positions

- **Anti-Modern DM Strength:** วิจารณ์ว่าการวัด Strong/Weak ในสำนักสมัยใหม่มักสร้างความสับสนและไม่สมบูรณ์ — เสนอให้เน้นที่ "คุณภาพ" ของ Month Commander แทน
- **Officer as Nobleman:** เน้น "Secret Technique" ว่า Officer star สามารถเป็น 贵人 (Nobleman) ได้ถ้าเลือก Useful God ถูก — ไม่ใช่ 小人 (Petty People) เสมอไป
- **Qi Flow ≠ Voodoo:** ถ้าพูดถึง Feng Shui/Qi โดยไม่มี "vertical Qi map" ที่วัดได้จริง = วูดู; เน้นการวิเคราะห์ Qi ที่ empirical มากกว่า
- **ต่อต้านพาณิชย์:** มองวิชานี้เป็นการให้ทาน ไม่ใช่ธุรกิจ; วิจารณ์ [[Joey Yap (杨登亮)|MsAocm]] ว่า "money-squeezing"
